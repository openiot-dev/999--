4. 손님은 갔는데 에어컨은 새벽까지 ON. 전기료 누가 냅니까?

> 예약·결제·입실·정산·청소까지, 오토플레이스가 알아서 돌립니다.
> 신용카드 없이 무료로 시작 👉 autoplace.openiot.app

### 본문 카피 — 중간 버전

> "자리를 비워도 매장은 돌아갑니다."
>
> 심야에 도어락 비번 안내하고, 여러 플랫폼 예약 맞춰보고, 손님 나간 뒤 조명 끄러 가고, 월말엔 며칠씩 정산하고…
> 이거, 사장님이 직접 하면 안 됩니다. 시스템이 해야 됩니다.
>
> 오토플레이스는 예약→결제→입실→이용→퇴실→정산·청소까지 6단계를 자동으로 돌립니다.
> - 다채널 예약 통합 + 한 곳 예약되면 나머지 자동 마감 (중복예약 끝)
> - 예약 시작 시각에 도어락·조명·냉난방 자동 ON, 끝나면 자동 OFF
> - 상황별 카카오 알림톡 자동 발송, 도어락 임시 비번 자동 발급
> - 매출·경영지표 대시보드 + 토스 지급대행 자동 송금
>
> 신용카드 등록 없이 무료로 시작하세요. 👉 autoplace.openiot.app

### 본문 카피 — 긴 버전

> 솔직히 하나만 물어볼게요. 지난 토요일 밤 11시에 뭐 하고 계셨어요?
>
> 휴대폰 붙잡고 손님한테 도어락 비번 알려주고, "들어가셨어요?" 확인하고 있지 않으셨나요?
> 무인 매장이라면서요. 근데 정작 무인이 아닌 건 사장님 본인이잖아요.
>
> 무인 공간은 빠르게 늘고, 인건비는 매년 오릅니다. 이런 시대에 운영을 손으로 돌리는 건 선택이 아니라 손해예요. 자동화는 이제 생존의 문제입니다.
>
> 오토플레이스가 사장님 대신 일합니다.
> ① 네이버·스페이스클라우드·에어비앤비 예약을 한곳에 모으고, 한 채널이 예약되면 나머지를 자동 마감해 중복예약을 막습니다.
> ② 예약확인부터 사전안내·시작·종료·정산까지 상황별 카카오 알림톡 17종을 자동 발송하고, 도어락 임시 비밀번호도 자동으로 발급합니다.
> ③ 예약 시작 시각이 되면 도어락이 열리고 조명·냉난방·노래방이 켜지고, 종료 시각엔 알아서 꺼집니다. (전기료 낭비·안전사고 걱정 끝)
> ④ 객단가·시간 활용률·이익률까지 보이는 대시보드, 토스 지급대행 자동 송금, 다점포 통합 통계로 월말 정산이 클릭 한 번으로 끝납니다.
> ⑤ 청소 매니저 자동 배정 + 청소 전후 사진 확인까지.
>
> 한마디로, 월급 안 받고 24시간 일하는 무인 매니저를 들이는 겁니다.
> 기존 예약 플랫폼은 그대로 쓰셔도 되고, 데이터 이전 작업도 필요 없습니다.
>
> 이제 휴가 가서도 매장 걱정 없이 쉬세요. 시스템에 모두 맡기세요.
> 신용카드 등록 없이 무료로 시작 👉 autoplace.openiot.app

## (d) 페인포인트 → 해결 베네핏 3쌍

**1. 심야에도 사장이 직접 입실 안내**
😩 토요일 밤 11시, 손님한테 도어락 비번 알려주고 입장 확인하느라 못 쉽니다.
✅ 예약 시작 시각에 도어락 임시 비번 자동 발급 + 카카오 알림톡 자동 안내. 사장님은 주무시면 됩니다.

**2. 플랫폼마다 흩어진 예약 → 중복예약·매출 누락**
😩 네이버·스페이스클라우드·에어비앤비… 일정 맞추다 중복예약 나고 매출이 샙니다.
✅ 다채널 예약을 한곳에 통합 수집하고, 한 채널 예약 시 나머지를 자동 마감. 중복예약 걱정 끝.

**3. 손님 퇴실 후에도 켜진 조명·냉난방 → 전기료·안전 위험**
😩 손님은 갔는데 에어컨·조명은 새벽까지 ON. 전기료 새고 화재 걱정까지.
✅ 종료 시각에 도어락·조명·냉난방 자동 OFF (SmartThings·헤이홈 연동). 낭비도 위험도 자동 차단.

## (e) 반론처리 카피 3개

**"설치가 복잡하고 부담되지 않을까요?"**
신용카드 등록 없이 무료로 시작할 수 있습니다. 기존 예약 플랫폼은 그대로 쓰면서 병행할 수 있고, 데이터 이전 작업도 필요 없습니다. 부담 없이 켜보고, 맞으면 그때 유료 플랜으로 넘어가세요.

**"지금 쓰는 예약 플랫폼을 계속 쓰고 싶은데요."**
바꾸실 필요 없습니다. 네이버·스페이스클라우드·에어비앤비 등 기존 채널을 그대로 두고, 오토플레이스가 그 예약들을 한곳에 모아 통합 관리합니다. 한 채널이 예약되면 나머지는 자동 마감되니, 플랫폼은 유지하면서 중복예약만 사라집니다.

**"비용이 부담돼요. 작은 매장인데 괜찮을까요?"**
무료로 시작해서 일정 사용량까지 무료로 쓰실 수 있습니다. 신용카드도 필요 없어요. 오히려 심야 응대 시간, 새는 전기료, 며칠씩 걸리던 정산 시간을 줄여줍니다. 1인·심야 운영 사장님일수록 효과가 큽니다. 필요해지면 Growth/Scale 플랜으로 올리세요.
