## 편2. 비포/애프터 — "휴가 가서도 매장이 돌아가는 이유"

| 시간대 | 화면/연출 | 자막(굵게) | 내레이션/대사 |
|---|---|---|---|
| 0:00–0:03 | 해변에서 선글라스 낀 사장이 음료 든 컷. 좌상단 'BEFORE/AFTER' 토글 | **이 사장님, 지금 발리에 있습니다** | "근데 매장은 풀가동 중이에요. 어떻게요?" |
| 0:03–0:09 | 화면 반 갈라 BEFORE: 폰 붙잡고 응대·정산하는 지친 사장 | **BEFORE: 휴가 와서도 폰만 붙잡음** | "예전엔 여행 와서도 예약 확인하고, 비번 알려주고, 정산하느라 못 쉬었죠." |
| 0:09–0:16 | AFTER: 오토플레이스 대시보드 — 다채널 예약 통합·알림톡·매출 그래프 | **AFTER: 시스템이 알아서 다 합니다** | "이젠 흩어진 예약이 한곳에 모이고, 한 채널에서 잡히면 다른 채널은 자동 마감. 중복예약 끝." |
| 0:16–0:22 | 종료 시각 조명·냉난방 자동 OFF + 토스 자동 송금 알림 | **퇴실하면 전기 자동 OFF, 정산도 자동** | "손님 나가면 불·에어컨 알아서 꺼지고, 며칠 걸리던 정산은 대시보드로 끝." |
| 0:22–0:28 | 사장 다시 해변에서 여유롭게 웃음 → 로고 | **월급 안 받는 무인 매니저, 24시간 근무 중** | "이제 시스템에 다 맡기세요. 신용카드 없이 무료로." |

**캡션:**
사장님 없어도 매장이 돌아가냐고요? 됩니다.
여러 플랫폼에 흩어진 예약을 한곳에 모으고, 한 채널에서 예약되면 나머지는 자동 마감. 입실·퇴실 IoT 자동 제어, 퇴실 후 전기 자동 OFF, 토스 지급대행으로 정산 자동 송금까지.
오토플레이스는 월급 안 받고 24시간 일하는 무인 매니저예요. 휴가 중에도 매장 걱정에 못 쉬던 사장님, 다음 휴가는 진짜 '쉬는' 휴가로.

## 편3. 숫자/시연 훅 — "예약 1건 = 자동으로 벌어지는 6가지"

**영상 길이:** 약 32초 | **포맷:** 세로 9:16

| 시간대 | 화면/연출 | 자막(굵게) | 내레이션/대사 |
|---|---|---|---|
| 0:00–0:03 | 폰에 '예약 1건 접수' 알림 '띵!' + 카운터 '0→6' 애니메이션 | **예약 1건 들어왔습니다. 손 까딱 안 해도** | "지금부터 자동으로 벌어지는 일, 6가지 셉니다." |
| 0:03–0:07 | ① 다채널 화면에서 타 채널 '마감' 도장 '쾅' | **① 다른 채널 자동 마감 (중복예약 끝)** | "한 곳에서 예약되면 네이버·스페이스클라우드 자동 마감." |
| 0:07–0:11 | ② 카카오 알림톡 자동 발송 + ③ 도어락 임시 비번 발급 | **② 알림톡 자동 발송 ③ 임시 비번 발급** | "예약 확인부터 사전 안내까지 알림톡이 알아서, 비번도 자동." |
| 0:11–0:16 | ④ 시작 시각에 도어락 열림·조명·냉난방·노래방 ON 몽타주 | **④ 시작 시각 도어락·조명·냉난방 자동 ON** | "시작 시간 되면 문 열리고 불 켜지고, 손님은 폰으로 직접 제어." |
| 0:16–0:21 | ⑤ 종료 시각 전체 OFF + ⑥ 대시보드 매출·정산 자동 집계 | **⑤ 종료 시 자동 OFF ⑥ 매출·정산 자동 집계** | "끝나면 다 꺼지고, 매출이랑 정산은 대시보드에 척." |
| 0:21–0:27 | '6/6' 완료 체크 + 사장 뒷짐 지고 흐뭇 → 로고 | **사장님이 한 일: 0개** | "예약 1건에 6가지가 자동. 사장님은 아무것도 안 했어요." |
| 0:27–0:32 | CTA 화면 | **신용카드 없이 무료로 시작** | "이거 모르면 손해. 지금 바로 시작하세요." |

**캡션:**
무인 매장이 폭발적으로 느는 지금, 예약 1건 들어오면 오토플레이스가 자동으로 하는 일 6가지:
① 타 채널 자동 마감(중복예약 방지) ② 카카오 알림톡 자동 발송 ③ 도어락 임시 비번 발급 ④ 시작 시각 도어락·조명·냉난방 자동 ON ⑤ 종료 시각 자동 OFF ⑥ 매출·정산 자동 집계.
예약→결제→입실→이용→퇴실→정산·청소까지, 사장님 손은 0번 갑니다. 이제 시스템에 다 맡기세요.
