# Part 1. 핵심 플레이북(도메인별 종합)

> 반복 핵심을 도메인별로 중복 제거·재구성. 세부·원문 뉘앙스는 Part 2 참조.

## 채널별 실전 마케팅 전략

각 채널의 공식·단계·전술·수치·툴·알고리즘·후킹 예시를 채널별로 통합 정리. 모든 플랫폼 알고리즘의 본질은 **반응도 하나**다(아래 공통 원리 참조).

---

#### 공통 원리: 반응도(알고리즘) 와 4사분면(채널 선택)

**반응도 공식 (모든 플랫폼 공통)**
> 반응도 = 체류시간 + 클릭(사진·메뉴·전화번호) + 공유 + 리포스트/저장 + 댓글 + 구독/팔로우
- 플랫폼이 다르더라도 "좋은 정보 → 사용자 체류 → 광고 수익" 구조는 동일. 반응도가 높을수록 알고리즘이 밀어준다.
- 네이버지도: 사진 → 체류 → 메뉴·전화번호 클릭 → 공유 = 상위 노출
- 유튜브: 썸네일 클릭 → 시청 지속 → 댓글·공유·구독 = 알고리즘 확산
- 쓰레드: 리포스트 + 공유하기 수치가 핵심 지표
- 표본·반응도·피카소 이론은 네이버지도/온라인 마케팅/릴스 모두 동일 적용 (출처: 릴스 초대박, 빵원 릴스)

**4사분면 마케팅 매트릭스 (채널 자동 결정)** (출처: 4사분면만 이해하세요, 온라인마케팅 2시간, 1억 이상 트렌드)
- X축(도달범위): 지역 ↔ 전국 / Y축(사업속성): 정보 비대칭 ↔ 이미지

| 사분면 | 예시 업종 | 주력 채널 |
|---|---|---|
| 지역 + 정보 비대칭 | 카센터, 지역 피부과, 인테리어, 심리상담, 세무사, 학원 | 네이버 블로그 |
| 지역 + 이미지 | 지방 맛집·카페·북카페·마사지·미용실·사진관 | 네이버 지도(플레이스) + 블로그 |
| 전국 + 정보 비대칭 | 마케팅 회사, 이별 상담, 출판사, 중고차 | 유튜브 롱폼 + 릴스 |
| 전국 + 이미지 | 호텔, 전시회, 캠핑장, 온라인 쇼핑몰, 로고 디자인 | 릴스·쇼츠·숏폼 + 퍼포먼스 마케팅 |

- **규칙:** 지역 → 검색 기반(블로그·지도) / 전국 → 불특정 다수 노출(릴스·유튜브). 정보 비대칭 강할수록 블로그 병행.
- **경계 사례(강남·청담 등 유동인구 많은 상권):** "지역과 전국 중간" → 블로그 + 릴스 병행.
- **주력 하나에 집중, 나머지는 보조('잔잔발리')**. 모든 채널에 한두 개씩 깔아두면 채널당 하루 1명 → 6채널 = 하루 6명 신규 고객.

**마케팅 트렌드 자가계산 공식** (출처: 온라인마케팅 2시간, 1억 이상 트렌드)
> "사람이 하루에 일어나서 가장 많이 접촉하는 시간·공간 = 그 시대의 마케팅 전쟁터"
- 1950s 신문·간판·입소문 → 1990s TV → 2006 네이버(배너 1시간 3천만원) → 2012 스마트폰 → 2019~20 유튜브 롱폼 → 2024~ 숏폼 → 2025~ AI 에이전트
- 수치: 틱톡/릴스 사용시간 2019.8월 611억분 → 2024.8월 약 1,222억분(2배) / 인스타 1.8억분 → 3.2억분 / 네이버앱·카카오톡 감소

#### 네이버 블로그 (+ CUE 알고리즘)

**2026 블로그 구조 전환 (지수 폐지 → 홈판)** (출처: 2026년 네이버 블로그 전성기)
- 구 구조: 블로그 지수(최적화/준최적화/저품질) → 오래된 고지수 우대, 신규 진입장벽 높음
- 신 구조: **네이버 홈판 알고리즘** → 게시물 품질 기준 노출, 신규 블로그도 동등 경쟁
- 신규 블로그 글 2~3편 수준도 홈판 노출 → 수만 조회수 가능. 구독자 사례: 시작 직후 월 400만원 수익.

**CUE 알고리즘 작동 원리** (출처: 2024년 네이버 블로거 바사삭)
- 자연어 질문 입력("일요일 양평에서 데이트, 6시간 맛집·카페 일정 짜줘") → AI가 단일 키워드 대신 연관 키워드를 연쇄 탐색 → 결과 합산.
- CUE는 창의적 추론이 아니라 **블로그·지도에서 가장 많이 언급·신뢰도 높게 쌓인 콘텐츠**를 우선 노출 = 기존 SEO의 연장선.
- 노출 결정 3요소: ① 블로그 포스팅 여부·품질 ② 네이버 지도 최적화 ③ 신뢰도 높은 블로그들의 언급 빈도.
- CUE 확산 타임라인: 1년차(사용률 낮음) → 2~3년(기존 검색과 병존) → 5~10년(음성/모바일 연동 완전 전환). 초기 주 사용층은 어린 연령층.

**CUE 대응 실행 단계**
1. 기존 블로그 최적화(핵심 키워드 본문 반복) 유지
2. **사용자 의도 키워드 추가 발굴** — 업종·장소의 '속성'으로 검색할 문장/단어 5~10개 추출
   - 북카페: 루프탑, 조용한, 사람 없는, 힐링, 셀카존, 감성 / 맛집: 맵지 않은, 간이 적절한 / 치과: 안 아픈, 치료 기간 짧은
3. 키워드를 본문에 자연스럽게 1~3회 삽입(스패밍 금지)
4. 제목·소제목에 자연어 검색 문장 반영 ("강남에서 조용하고 사람 없는 카페 찾아줘")
5. 네이버 지도 병행 최적화
6. 체류시간 높은 고품질 콘텐츠 작성
- 실증: '욕망의 북카페' → "강남에 있는 북카페 추천해줘" CUE 검색 1번 노출 (블로그 다수 + 지도 최적화)

**자청 표준 블로그 글 구조** (출처: 직원 0명 AI, 압구정 병원마케팅)
1. 마인드 리딩(공감) 도입부 — 독자 불안·고민 2가지 먼저 언급 (예: "동탄 핸드폰 매장 그냥 사지 마세요" → 싼지/사기인지)
2. 가치 입증 — 연수·판매량·비포애프터 사진(숫자의 원칙)
3. 구체적 증거(리뷰·사례·수치, 반박 불가 형태)
4. CTA — 연락처·링크·사진 위치 안내
- 순서 원칙: 상대방이 궁금한 것(비포애프터·후기·경력) 먼저 → 전문성 제시. **팔짱 낀 경력증 나열형 금지**(신뢰도 급락).

**브랜드 블로그 돈 버는 공식 2가지** (출처: 온라인마케팅 2시간)
1. 키워드 전략: 정보 탐색 키워드(아직 지갑 안 열림) vs 구매 직전 키워드("한 놈만 걸려라")
2. 구매 동기 + 반박 제거: 타겟이 가장 듣고 싶은 말 보여주기 → 의심·망설임 제거
- "지식의 저주" 주의: 전문지식 용어를 중학생도 이해하게. 1일 1포스팅보다 퀄리티가 중요.

**셀링 포인트 입력 공식 (AI/직접 공통)** (출처: 직원 0명 AI)
> 업체명/브랜드명 + 잡고 싶은 키워드 + 셀링 포인트(구체 수치)
- 셀링 포인트는 반드시 연수·건수·비교로: "한우곰탕 32년, 다른 집 1~2일 숙성, 우리는 한 달 이상 숙성". **추상어 절대 금지**("압도적", "뛰어난", "최고의").

**블로그 3대 실패 케이스:** ① 일기 ② 단순 상품 리뷰 ③ 광고·이벤트만 홍보. 여행기록·맛집리뷰 혼재 금물. (출처: 온라인마케팅 2시간)

**블로그 카피·제목 예시**
- 휴대폰 매장 첫 문장: "문의가 빗발치는 중입니다. 문의 전화가 많아 현재 통화 중일 수 있습니다." (희소성·사회적 증거) (출처: 3가지 이론)
- 독서실 제목: "수학지도 30년 상주 선생님이 계신 곳" / 첫 줄: "수학지도 30년 경력 대표님이 ○○에서 자문까지 받았대" (출처: 3가지 이론)
- 병원 블로그 제목: "[지역명] + [증상/시술] + 환자가 실제 질문한 문장" → "남양주 테니스 엘보 몇 개월 쉬어야 되나요" (출처: 2025 병원마케팅)
- BAD→GOOD: "제이쌤이 행복한 이유는" → "상대 영어학원 확실한 성적 향상 사례를 확인해 보세요"(44점→90점 성적표) / "스피닝하고 횡문근 융해증"(지식의 저주) → "원주 헬스 이거 모르고 등록하면 100만원 날린다. 20년차 물리치료사…" (출처: 온라인마케팅 2시간)
- 제목 후킹(AI 생성 우수): "유명하다고 무조건 등록하지 마세요"(언매칭) / "아직도 저렴한 곳만 찾으세요?"(역공감) / "이거 확인 안 하면 후회합니다" (출처: 직원 0명 AI)

**블로그 수치/사례:** 병원 블로그 단독으로 6개월 만에 순수익 5,000만→1억 / 동탄 휴대폰 매장: 월 700만원(3년 정체) → 블로그·체험단 후 월 3,000~4,000만원 / 이상한마케팅 창업 초기(2018) 브랜드 블로그로 하루 문의 15~20건 / 아트라상: 블로그 마케팅 0원 투자로 월 순수익 1억~1억5천 / 물리치료사 원장: 직원 0명·블로그 중심 → 월 순수익 1억.
**파괴력:** 과거 100 → 현재 30~40 수준으로 감소했으나, AI 검색 추천의 기반 데이터로 작용하므로 기본값으로 반드시 해야 함.

**블로그 체험단(프블단/포일단):** 일반 체험단(5,000~15,000원, 크몽 30만원대) 금지 — 품질 담보 불가. 전문 트레이닝 작가(180~200명, 입사 후 하루 10시간 글쓰기 3개월 인턴 통과) 활용. 네이버 검색 결과 5개 중 3~4개를 자기 클라이언트 글로 채우는 수준 목표. 첫 줄부터 신뢰 근거 제시, 광고티 나면 탈락. (출처: 3가지 이론, 5개만 해봐)

#### 네이버 지도/플레이스 (클릭률·반응률·활용력)

**상위 노출 3요소** (출처: 네이버 지도 세가지)
1. **클릭률** — 검색 목록에서 내 업장을 클릭하게 하는 힘 (대표 사진 품질이 핵심)
2. **반응률** — 진입 후 사진 클릭·전화·예약·내비·저장·공유·체류 등 행동량
3. **활용력** — 네이버 제공 기능(예약·톡톡·쿠폰·소식·메뉴 이미지·정보 상세) 활용도

**클릭률 높이기**
- 대표 사진(1~3번)은 고품질로 교체 — 아이폰 색보정 또는 전문 작가. **갤럭시 무보정 탈락, 색보정 필수**.
- 사진은 검색 의도에 맞게 (북카페 검색자 → 조용한 인테리어 / 루프탑·빵 사진 X)
- 사진은 4~5장이 아니라 최대한 많이, 정보성 있게.

**반응률 높이기**
- 메뉴 항목에 이미지 + 차별화 카피 ("국내 최초 디카페인 전문 로스터리에서 공수받은 원액기")
- 방문자·블로그 리뷰 유도, 정보 비대칭 해소로 상세정보 풍부하게 → 신뢰도 ↑ → 전화/방문 ↑
- 리뷰에 반드시 답글 + 다음 방문 혜택 제시

**활용력 높이기**
- 톡톡 연동 / 네이버 예약 연결(필수, 톡톡보다 더 강력) / 네이버페이 / 쿠폰 등록
- '소식' 탭 주 1회 이상 공지·이벤트 업데이트
- 정보(소개글) 섹션에 글 최대한 많이 + 타깃 키워드 2~3회 자연 노출 + 수치 포함("18년 경력")
- 얼굴 사진 포함 프로필, 진료/영업시간 정확 입력, 휴진 공지, 실내 사진
- **대표 키워드 설정 시 지역명 불필요** (위치는 이미 고정값)
- 유입량: 내부(플레이스 광고) + 외부(당근·인스타·메타 광고)
- 최신성: 리뷰·소식글·사진·영상·클립·저장수·스마트콜·톡톡·예약·페이 최소 월 1회 업데이트
- 투자 시간: 주말 1시간이면 기본 적용 가능

**지도 확장 (네이버 지도가 잘 될 경우):** 카카오 맵 + 티맵 + 구글 맵 추가 최적화. 단, 카카오 맵 평점 2.5면 신뢰성 즉시 하락. (출처: 1억 이상 트렌드)

**알고리즘/원리:** 네이버는 '기능을 적극 활용하는 똑똑한 사업자'를 가산. 핸드폰 GPS 데이터로 어뷰징 탐지 가능(봇은 방문 안 함 → 위치 불일치). 대부분 경쟁사가 기본도 안 하므로 1~2개만 적용해도 상위 노출 가능. 강남 등 초경쟁 상권은 추가 트래픽·광고 5~10% 기여.

**수치/사례:** 욕망의 북카페 — 서울 북카페 검색 200개+ 중 1등 / 장중진 세무사 — 얼굴+책 저자 사진 + 톡톡으로 포천시 세무사 압도적 1등 / 포천 세무사 — 예약·톡톡·얼굴사진 등록한 1위가 정보 없는 2~6위 대비 압도적 클릭률.

**카피:** 메뉴 상세 "국내 최초 디카페인 전문 로스터리에서 공수받은 원액기" / 플레이스 광고 문구 "1등급 성적 향상 1천 건 이상".

**주의:** 어뷰징/불법 트래픽 구매 절대 금지(봇 100명 진입→즉시 이탈→비정상 이탈률 탐지→노출 차단/강등/삭제) / 사진 4~5장만 올리고 방치 금지 / 검색 의도와 안 맞는 사진 금지 / 메뉴·정보 빈칸 방치 금지 / 소식 탭 업데이트 중단 금지.

#### 숏폼 (릴스·쇼츠·틱톡)

**채널별 사용자 성향:** 릴스(인스타) = 상대적 고지능·정보성 반응 / 쇼츠(유튜브) = 좀 더 가벼운 층 / 틱톡 = 10대·글로벌·재미. 국내 사업은 릴스·쇼츠가 효율적, 틱톡은 해외/10대. (출처: 2024 트렌드, 무료 숏폼)

**자영업자 숏폼 4단계 시스템** (출처: 무료 숏폼)
1. 릴스로 무료 마케팅 — 주 1개씩 꾸준히, 1만 뷰짜리 30개 = 누적 300만 노출
2. 릴스 → 유튜브 쇼츠 크로스포스팅 (추가 공수 없이 노출 2배, 플랫폼이 신규 기능 밀어주는 알고리즘 이용)
3. 행동 유도(CTA) + 스토리텔링 — 캡션에 창업 스토리·차별화 이유 12~15줄, 전 게시물에 복붙(항상 처음 온 사람 기준)
4. 인스타 스폰서 광고 A/B 테스트 — 장점 1개씩 쓴 소재 4개를 각 5,000원씩 → 효율 좋은 소재 집중
- 자영업자는 유튜브 롱폼 불필요, 쇼츠만. 7장면 3초씩 + 템플릿 → 퀄리티보다 업로드 빈도 우선.

**릴스 초간단 템플릿 2단계 (왕초보)** (출처: 릴스 100만 왕초보)
1. 인스타 앱 + 버튼 → 우측 상단 '템플릿' → 선택 → '미디어 추가'(최대 10개) → 다음 → 공유
2. 잘 되는 릴스 분석 → 장면·편집·트렌드 녹여내기
- AA 버튼으로 호기심 자극 문구 추가, 커버(썸네일) 수정, 해시태그는 대표 키워드 1~2개면 충분(#오사카여행 #서울카페). 마음에 드는 남의 릴스 → 템플릿 크레딧 클릭 → '템플릿 사용'. 좋은 템플릿은 … → 저장 후 재사용.
- 릴스 > 사진(노출 압도적 유리). 조회수 2,000~3,000도 사업 홍보로 충분. 팔로워 50만~100만도 전략 없으면 10만 미만 흔함.

**자청의 숏폼 4가지 공식** (출처: 빵원 릴스)
1. **표본 이론** — 카피·썸네일이 반응할 잠재 독자 수를 최대한 넓혀라. "사이코패스 거르는 5가지 책"(50만 명) → "김경일 교수가 추천하는 5가지 책"(전국민) = 100배 차이
2. **갈라치기** — 두 집단(A vs B) 비교, 한쪽 손해/한쪽 이득. "MZ세대는 ○○하고 / 기성세대는 ○○한다"(라임 맞춤)
3. **궁금증 유발** — 결론을 보여줄 듯 말 듯 끌기(2~3초 기본 → 5초 이상 지연). 불편함 축적 → 해소 순간 도파민
4. **언매칭** — 상식과 정반대 단어 결합. "재회하고 싶으면 매달리지 마세요" → "재회하고 싶으면 쌍욕을 하세요"(60만 뷰)

**릴스 초대박 3가지 공식** (출처: 릴스 초대박, 발리 240만뷰)
1. **피카소 이론** — 창의 말고 잘된 콘텐츠의 형식(포맷)을 먼저 모방, 내용은 내 것 (베끼기 ≠ 형식 차용)
2. **표본 이론** — 표본 넓은 주제(GPT 거짓말 500만명, 부자 되는 법, 중학생도 할 수 있는 ChatGPT) vs 좁은 주제(프롬프트 잘 만드는 법 1~2만명, 미드저니, LLM 개발)
3. **반응도 이론** — 시청 지속·공유·댓글·좋아요·저장·DM을 기획 단계부터 설계. (예: 릴스는 복붙 안 됨 → 캡션/댓글 확인 유도 → DM 응답으로 프롬프트 제공)
- **채널 초반 팔로워·지인 공유 요청 금지** (관심 없는 팔로워는 알고리즘 악영향)

**잘 터지는 릴스/쇼폼 18가지 공식** (출처: 온라인마케팅 2시간)
1. 분석 어셈블(첫 5초 음악·장면·문구·감정 쪼개기, 가장 중요) 2. 숫자의 원칙 3. 특이한 장면 4. 유행 음악(1~2개월마다 교체) 5. 업매칭(상반 조합) 6. 박자 맞추기 7. 1초의 법칙 8. 잘못 봤나("어? 뭐야?") 9. 스토리텔링(기승전결·감정곡선) 10. 감성 11. 논란거리 제공(댓글 싸움 유도, 고급) 12. 언매칭 13. 귀여움 14. 특이한 배경(판테온·콜로세움) 15. 표본 이론 16. 가치 깎기(가치 높은 사람의 실패·슬픔) 17. 꼬아주기(유행 보여주다 "절대 하지 마세요" 반전) 18. 팔로우 이유(삶에 도움)

**숏폼 제작 실전 디테일** (출처: 온라인마케팅 2시간)
- 레퍼런스 전용 비즈니스 계정 → 같은 업종 인플루언서 팔로우 → 100만 조회 영상 100~200개 분석(카피라이팅 + 영상 전개)
- 도입부 3초 후킹: 시각(첫 장면) + 청각(소리) 동시 자극
- 썸네일 카피 글자수 15자 이내
- **인스타 데드존:** 상단 300px·하단 400px 텍스트 배치 금지
- 한 장면 3초 이하 빠른 전환
- "뭐뭐하지 말고 이거 하세요"(명령형) > "이렇게 하면 망하더라고요"(서술형)
- 반박 제거: 업종·제품 의심 포인트를 이미지·카피로 선제 해소

**숏폼 카피 응용 예시:** "절대 짜면 안 되는 피부 트러블 3가지"(병원) / "세금 폭탄 맞는 3가지 상황"(세무사) / "인테리어 모르면 몇 천만원 날리는 디테일 5가지" / "예적금하지 말고 이거 하세요" → "재회할 때 매달리지 말고 이거 하세요" / "1등 하면 생기는 일"(역행자 책, 궁금증 유발) / "내 여자친구 사랑하면 데려가야 할 맛집"(스테이크집, 공유 유발) / "GPT 거짓말 잡아내는 초단법 / 너 그냥 이거 넣기만 하면 돼"

**스폰서 광고 문구(셀프 사진관):** "사진사 없어서 부담 없음" / "일반 스튜디오 대비 30% 저렴" — 각각 별도 소재 5,000원씩 테스트. 디자인 해치더라도 장점 문구 필수.

**수치/사례:** 자청 북카페 릴스 6.8만 뷰(7장면 3초+템플릿) / 아울 인테리어 팔로워 1만→5만(두 달), 월 수주 30~100개 / 그레이그라피 하루 5,000원 광고로 고효율 / 발리 릴스 이틀 240만 조회·팔로워 4,000명(저장 3.6만·공유 2.3만·좋아요 1.2만·댓글 6천) / 지우 님 팀 100만 조회 100개+ / 역행자 책 릴스 22만(템플릿) / 게스트하우스 15만 / 익선동 고양이 16만 / 초음파 세척기 쇼폼 → 월 매출 1억 / 키즈 풀빌라 펜션 25만+ → 1일 만에 예약 20건 / 파주 사업장 릴스 100만 → 1개월 매출 1억 / 60만 조회수 릴스 제작 5분.

**주의:** 자영업자 유튜브 롱폼 시작(비용 대비 낮음, 쇼츠 크로스포스팅으로 대체) / 한두 번 테스트 후 포기 / 과도한 공수 / 조회수 집착(매출 연결이 기준) / 표절(레퍼런스 분석·응용 OK, 복붙은 알고리즘에도 안 먹힘) / 첫 문장 지역명 좁게("창원 가면~") → 표본 박살.

#### 카카오 숏폼

(출처: 지금 당장 카카오 숏폼)

**플랫폼 초기 진입 법칙**
> 공급 < 수요인 시점 진입 → 동일 실력으로도 조회수·수익 훨씬 큼 (낮은 공급 × 높은 수요 = 초기 진입자 독식)
- **논란 = 진입 기회:** 사람들이 시끄러울 때가 공급이 가장 적은 시점. 기대값 사고(확률×보상)로 "기대값적으로 무조건 해야 된다"고 판단.

**실행:** 지금 당장 계정 개설·업로드 / 팀·직원 단위 채널 분산 운영(의무화) / 카카오 쇼핑(카카오톡 간편결제) 연동 상품 판매 구조 미리 설계.

**핵심 인사이트:** 카카오톡 결제 인프라로 전환율 높을 것으로 예상 / 플랫폼은 논란 후 익숙해지는 시간을 기다렸다 전면 밀어붙임 → 그 전에 선점 / 유튜브·쓰레드·릴스·인스타 모두 초기 진입자가 성공한 반복 패턴.

**수치:** 자청 회사 한 팀이 카카오 숏폼 초기에 조회수 17만 / 인스타 릴스도 초기 셀럽 반발에도 강행 → 성공.

**카피:** "지금 카카오 논란인데 똑똑한 사람들은 지금 오히려 기회라고 생각" / "공급은 적고 수요가 많으면 때돈을 법니다" / "결국 언젠간 카카오 숏폼도 보고 있게 될 거예요"

**주의:** 논란·불확실성 이유로 관망 금지(논란 = 공급 감소 시그널) / 완전히 자리잡은 뒤 진입하면 늦음(레드오션).

#### 유튜브 / 롱폼

**잘 되는 콘텐츠 역공학 3단계** (출처: 5분만에 조회수 3배)
1. 내 채널 카테고리 키워드(영문법, 영어 단어, 영어 독해)로 유튜브 검색
2. 필터 → 조회수순 / 길이 4~20분 / 전체 기간
3. 상위 콘텐츠의 썸네일·제목 패턴(숫자·해결·시간 단위) 추출 → 내 식으로 변형 ("주제를 베끼는 것이지 영상을 베끼는 게 아니다")

**유튜브 콘텐츠 발굴 공식(병원 등)** (출처: 2025 병원마케팅)
1. 유튜브에서 진료 키워드 검색 → 2. 우측 상단 '조회수' 정렬 → 3. 잘 나온 영상 포맷·썸네일 분석 → 4. 내 지식으로 동일 주제 재해석 촬영
- 표본 낮은 전문 영상(조회수 1,000~5,000) 목표 → 진상 필터링·의료법 신고 회피. 허리통증 같은 광범위 주제 X, 눈밑지방·코필러 등 특정 시술 집중.

**숫자의 원칙** (출처: 5분만에 조회수 3배, 2025 병원마케팅)
- 제목/썸네일에 구체 숫자 필수: 기간(10분·15분·3주), 점수 변화(44점→91점), 배수(3배·10배), "8분 만에", "99% 좋아졌어요"

**영상 초반 4단계 공식 (오프닝 10~20초)** (출처: 유튜브 조회수 3배 4가지)
| 단계 | 정의 |
|---|---|
| 1. 후킹 | 시작 2~5초 내 놀라운 장면/질문/궁금증으로 집중도 확보 |
| 2. 가치 입증 | 화자가 말할 자격 증명(성과·경력·수치). 썸네일로 대체 가능 |
| 3. 라포르(친밀감) | "나도 너와 같은 사람이야"(흙수저·실패 경험)로 거리감 해소 |
| 4. 도파민 | 새롭고 특이한 것으로 궁금증·놀라움 유발 |
- 한 장면에서 4단계 동시 충족 가능. 후킹은 맨 앞·영상 내부 필수. 지식 채널은 전문 용어 1~2개 전략적 삽입, 웃음·유머는 라포르 도구. 우울·무기력 주제는 2~3년 후 검색 누적되는 장기 콘텐츠.

**유튜브 갈래:** A) 4단계 공식(지식·인터뷰, 일반인 가능) B) 순수 도파민(몰카·브이로그·세계여행, 재능·외모 필요 → 일반인 비추)

**롱폼 구성 (풀링 vs 키)** (출처: 온라인마케팅 2시간)
- 풀링(Pulling): 표본 넓음·정보 탐색용·조회수 높음 / 키(Key): 표본 좁음·구매 직전용·매출 기여. 비율 풀링 4 : 키 1(또는 3:1). 도입부 후킹 10초 기준.
- 채널 세팅: 페르소나 → 채널명(개인 브랜딩이면 닉네임, 사업체면 사업체명) → 소개글에 블로그 링크. 손실 위협 카피("안 하면 20% 가산세") 효과적. 이유 4가지 중 2가지 삐리 음소거(궁금증). 블로그 글 = 유튜브 대본으로 그대로 활용. 썸네일 글자수 15자 이내.

**뷰트랩 3단계 활용** (출처: 뷰트랩 조회수 70만)
1. 영상 트렌드 검색 — 키워드 입력 + 필터(기여도·성과도: 굿/그레이트) → 알고리즘 타는 영상 추출
2. 채널 분석 — 채널 등록 → 굿·그레이트 + 쇼츠 제거 → 잘된 영상 30개 집중 분석 → 노션 아카이빙
3. 카피 변형 — 제목·썸네일 워딩을 그대로 X, 반드시 조금씩 변형
- 기여도(구독자 대비 폭발) vs 성과도(절대 조회수·구독자) 구분, 둘 다 굿·그레이트로. 노출 확률은 시의성 이슈(인물명) 있을 때만 참고. 자의식 해체("내 스타일 아니어도 대중이 원하면 쓴다"). 악플 증가 = 잘되는 신호.

**유튜브 제목·썸네일 공식**
- "[숫자]만 하세요" / "[유명인/권위자]가 말합니다"(자청이 아닌 "서울대 황농문 교수"가 말하는 형식 → 신뢰도↑) / 비포애프터(원으로 구분) / "충격적인 [형용사]+[명사]" / 순자산 숫자(5천억, 200억) 노출
- 통하는 제목 예시: "영어 어순 15분이면 해결된다"(342만), "10분 투자로 형용사 부사 평생"(29만), "영어 단어의 배신", "44점에서 91점 떡상", "30대 초반에 자동으로 월 1억씩 버는 방법"
- 의료: "누워서 3분씩 했더니 허리 통증이 99% 좋아졌어요", "8분 만에 허리통증 이별", "자면서 허리통증 고치세요"(360만 뷰). 공략 심리: 빠르게+쉽게+집에서+병원 안 가도 됨 중 1개 이상 썸네일에.
- 뷰트랩 사례: "가난한 사람의 충격적인 부자의 뇌"(62만, 구독자 200/일→2,000+/일), "이 책만 99번 봤습니다"(캘리체니 공식), "경찰 조사 이렇게 받으면 큰일납니다"(변호사 채널)

**오프닝 후킹 예시:** "유튜브로 성공하는 것은 실력일까요 운일까요? 165만 유튜버인 제가 아무것도 없는 무명 시절로 돌아간다면…" / "갑자기 꿈이 없는 사람은 개새끼예요" / "어떤 비난에 화가 난다는 건 다 내가 동의해 주는 거예요" / 세무사: "부모님이 돌아가셨으면 기간별로 꼭 해야 할 일이 있습니다. 안 하면 20% 가산세…"

**롱폼 인사이트:** 정보비대칭 업종(의료·법률·마케팅)에 필수 — 1분 숏폼으로 설득 안 됨, 5~10분 전문 영상이 전환율 압도. 롱폼은 보고 온 환자 = 신뢰·이해 완료(진상 확률 낮음). 한 번 올린 영상 3~6년 마케팅 효과 지속(장기 자산).

**수치/사례:** 자청 채널 평균 50만 vs 일반 마케팅 채널 2만 / 압구정 탈모병원 블로그·롱폼·쇼츠 병행 → 개원 2년 내 월 1.5억~2억 / 치과 원장 방문 고객 20%가 유튜브 보고 방문 / 세무사 수강생 구독 3명대→3만, 2024 매출 1억+ 증가.

**주의:** 1시간 강의 통편집 업로드 금지 / 자기 머리로만 제목 만들기 금지(조회수 100) / 완전 카피 금지(변형 필수) / 같은 공식 반복 남발 금지(표본 박살) / 화질·조명 완벽 기다리지 말 것 / 전문 용어 과다 금지(1~2개 적정).

#### 쓰레드 (Threads)

**쓰레드 마케팅 5단계** (출처: 10년하던 블로그 버렸습니다)
1. 부활 근거 2. 블로그 대비 효과 3. 알고리즘 원리(스아이/품마 절대 금지) 4. 조회수 5~10배 높이는 법(레퍼런스 분석) 5. 프로필 설정

**잘 터지는 5가지 원칙** (출처: 온라인마케팅 2시간, 10년 블로그)
1. **글자수:** 150자 이하, 한 줄 20자 이하(짧고 심플). 300자 제한.
2. **사진·영상:** 시각 정보 필수(체류시간 ↑, 피드 스크롤 멈춤)
3. **첫 문장:** 표본 넓게 + 구체 숫자 + 강조 표현. "저는 부자예요" < "저는 한 달에 1천만원씩 법니다" / "이거 좋다" < "인생책 1등, 최고" / "창원 가면 꽃가"(X) → "서울 놀러 가면 꽃가"(O)
4. **댓글 유도:** 본문에서 끝내지 말고 댓글로 이어지게 → 반응도 ↑. 댓글 스레드 300자×N개 = 장문 분할(체류시간 ↑)
5. **리포스트·공유 10개 이상:** 알고리즘 선택 기준. 댓글 100개여도 리포스트·공유 2개면 노출 낮음(균형 중요)

**레퍼런스 분석 루틴** (출처: 10년 블로그, 자청 필살공식)
1. 홈 피드에서 '리포스트 + 공유하기 10개 이상' 게시글만 필터
2. 30분간 해당 글만 저장
3. 첫 줄 / 마지막 줄 / 총 줄 수 / 엔터 위치 / 중간 흐름 / 주제 패턴 분석·필기
4. 형식(구조) 차용 → 내 콘텐츠 제작 → 반복 (분야별 공식 다름 — AI 분야 ≠ 맛집 분야 따로 분석). 주 1회 루틴화.

**프로필 설정 공식**
> [구체 숫자 + 행동량/실적] + [이득/전달 내용]
- "임플란트 9,754건 시술한 원장", "한 달에 300명씩 진료하는 의사", "일주일에 50개 카페 방문하는 사람이 추천하는 맛집", "70kg→43kg 감량한 [이름]". 추상적 표현("열심히 사는 사람") 금지.

**핵심 인사이트:** 쓰레드는 '상위 5~30%' 고지능 모수 집중 → 릴스 100만 뷰보다 쓰레드 5만 뷰가 마케팅 효과 클 수 있음 / 블로그는 키워드 타겟팅·안 퍼짐, 쓰레드는 알고리즘 확산 / 인스타가 쓰레드를 밀고 있음(피드·좋아요 탭 노출) / 공수 1/10(글 2~3분) / 정보성 콘텐츠만(상품 언급 금지 — 모텔 이론) / 일기장식 감성 글 금지.
- **링크 삽입 시 조회수 반토막~1/3토막** → 홍보 링크 본문에 넣지 말 것. 투마(상호 리포스트) 중노동 대비 매출 연계 안 됨.

**수치/사례:** 블랑시 치과 게시글 12,000회 / 스픽(팔로워 10만) 9만·3만 조회 / 자청 시작 2주 만에 리포스트 100개 돌파(12시간 내) / 리포스트 178+공유 245 = 전문가 모수 집중 신호 / 자청 익명 계정 54,000·32,000뷰 / 약 2.5개월 만에 팔로워 3.4만(탑 크리에이터).

**툴:** GPTs '자청 쓰레드'(초안 텍스트 붙여넣으면 쓰레드 형식으로 변환, ChatGPT '탐색'에서 검색).

**주의:** 스아이(SI)/품마 절대 금지(맞팔·좋아요·댓글 교환 → 반응 없는 계정으로 인식 → 노출 차단, 1만 팔로워 성장 불가) / 자기 멋대로 글 쓰기 금지(레퍼런스 먼저) / 댓글 100개=좋은 콘텐츠 착각 금지 / 처음부터 반응 기대 금지(10개+ 올리며 실행 근력).

### 2. 프롬프트 공식·기법

#### 2-1. 6가지 프롬프트 공식 (AI에게 일 시키는 법)
1. **역할 부여** → "너는 20년차 세계 최고의 블로그 마케팅 전문가야"
2. **목표 설정** → 업무 목적·배경 설명 ("네가 작성해 준 글을 읽은 잠재 고객이 매장에 방문하고 싶어져야 해")
3. **형식 지정** → 글자수·도입부 조건·말투 등 구체적 지시
4. **예시 제공** → 잘된 예시(Good/Bad)를 첨부 → 퀄리티 급상승
5. **깊게 고민(Chain of Thought)** → "천천히 생각하고 답변해 줘"
6. **동기 부여** → "이 업무는 회사 매출을 결정하는 아주 중요한 거야. 잘하면 100억 줄게, 못하면 혼날 줄 알아" (유료 프로 버전에서는 생략 가능)

(출처: 자청이 말하는 온라인마케팅 2시간만투자하세요)

#### 2-2. 구체적 질문 공식 (질문의 질 = 답변의 질)
- **공식**: 사업명 + 서비스 내용 + 운영 방식(온/오프라인) + 가격 + 매출 + 순이익을 모두 포함해 질문
- 막연한 질문("전략 짜줘")은 막연한 답변을 낳는다. 구체적 정보를 입력할수록 실행 가능한 조언이 나온다.
- **잘못된 예**: "지금 심리 상담하고 있는데 전략 짜줘"
- **올바른 예**: "지금 이별 상담 사업을 하고 있어. 이별한 커플을 심리학 이론으로 재회 확률을 높여주는 온라인 서비스야. 상담 가격은 [XX원], 매출은 [XX], 순이익은 [XX]야. 마케팅 전략 짜줘."

(출처: 상위 1% 일잘러만 아는 CHAT GPT)

#### 2-3. 황당하고 창의적인 전략 2개 요청 공식
- 일반 전략 외에 "황당하고 창의적인 전략 두 가지도 함께 제시하라"는 조건을 붙인다.
- 마케팅 전략은 10개를 던져 1~2개가 '얻어걸리는' 방식. 창의적·황당한 전략도 디벨롭의 씨앗이 된다.
- AI가 아이디어 15개 제시 시 쓸 만한 것 2~3개 도출 가능 → 한꺼번에 버리지 말고 선별 취득.

(출처: 상위 1% 일잘러만 아는 CHAT GPT; 나만을 위한 AI 5분만에)

#### 2-4. 2차 질문 연쇄 활용 (심화)
- GPT가 블로그 마케팅을 제안하면 → "세부 키워드 30~40개, 월간 검색량을 표로 그려줘"처럼 추가 질문으로 심화.
- 주의: GPT가 제시하는 월간 검색량 수치는 부정확할 수 있음 → 키워드 목록 발굴 용도로만 활용하고 정확한 검색량은 네이버 키워드 도구 등으로 검증.

(출처: 상위 1% 일잘러만 아는 CHAT GPT)

#### 2-5. 프롬프트 품질 개선 원칙
- 기본 생성(~300자) → 직접 수정·조건 추가 → 고도화(8,000자 수준). 첫 결과물에 만족하지 말 것.
- 결과가 마음에 안 드는 포인트를 찾아 마지막에 조건을 한 줄씩 추가.
- 짧게 서술된 조건(문화적 적합성, 문맥 유지 등)에는 반드시 예시 2~3개 추가 → 정확도 대폭 향상. 한 줄 이하 조건은 AI가 제대로 이해 못 함.
- **AI는 쉬운 언어일수록 효율적**: 초등학생에게 양자역학 설명하듯, AI가 이해하기 쉬운 'GPT 언어'로 작성해야 한다. GPT에게 프롬프트 초안 생성을 맡기면 이 언어 변환이 자동으로 이뤄진다.

(출처: 나만을 위한 AI 5분만에)

### 4. 콘텐츠 자동화 — 직원 0명 마케팅 (이상한 AI 솔루션)

이상한 마케팅 노하우를 학습시킨 전용 AI 솔루션으로 블로그·유튜브·인스타 콘텐츠를 마케터 수준으로 10~15분 안에 생성. 대행비(월 200~600만 원)를 월 10/20/55만 원 플랜으로 약 1/10~1/20 절감. (이상한 마케팅 아카데미 수강생 대상 판매)

#### 4-1. 콘텐츠 생성 3단계 입력 공식 (마이페이지 필수 입력)
1. **업체명/브랜드명** — 예: 아트라상, 동탄 폰 매장
2. **잡고 싶은 키워드** — 예: 이별 상담, 동탄 핸드폰 매장
3. **셀링 포인트(구체 수치 포함)** — "한우곰탕 32년, 다른 집은 1~2일 숙성, 우리는 한 달 이상 숙성" 식. 추상어 금지, 숫자·사실·비교로 작성

→ 3가지를 입력해두면 블로그·유튜브·인스타 어느 채널이든 주제만 지정하면 자동 생성.

#### 4-2. 자청 표준 블로그 글 구조 (AI 생성 적용)
1. **마인드 리딩(공감) 도입부** — 독자가 느끼는 불안·고민(2가지)을 먼저 언급
2. **가치 입증** — 연수·판매량·비포애프터 사진 등 숫자의 원칙으로 신뢰 구축
3. **구체적 증거 제시** — 리뷰·사례·수치(반박 불가 형태)
4. **CTA** — 연락처·링크·사진 위치 안내

#### 4-3. 핵심 원칙
- **숫자의 원칙**: "잘한다"는 0점. "10년간 3,000명, 비포애프터 100장"처럼 반박 불가한 수치로.
- **추상어는 마케터 실격**: "압도적", "뛰어난", "최고의" → 즉시 삭제. 자청이 유일하게 화내는 지점.
- **가치 입증 먼저, CTA는 그다음** — 신뢰 쌓기 전 구매 요청은 거부감만 유발.
- **AI는 사전 학습이 필수** — 일반 ChatGPT에 그냥 요청하면 추상어·방어 기제 유발 문장이 나옴. 전용 프롬프트/솔루션 필요.
- 제목은 AI가 10개 제시 → 쓸 만한 2~3개 선별.
- 사례: 핸드폰 대리점이 블로그 마케팅 시작 후 월 순수익 600만 → 다음 달 1,800만 → 현재 6,000만 원 이상.

(출처: 이제 직원 0명으로 마케팅 가능(AI))

#### 4-4. ChatGPT 단독 사용 시 나쁜 결과 (반면교사)
- 추상어 범벅: "압도적 판매량, 더 많은 직원, 더 나은 서비스"
- 반박 가능 도입부: "이곳에 한 번만 오시면 다른 매장을 찾으실 필요가 없습니다"
- 구체성 부족: "가격·서비스·사후 관리 모든 면에서 뛰어납니다"

### 6. 클로바노트로 회의록 10초 자동화

#### 클로바노트 + ChatGPT/Claude 2-툴 워크플로우 (3단계)
1. **녹음**: 클로바노트 앱으로 회의/상담/통화 녹음 (녹음 버튼만 누르면 됨, 별도 설정 불필요)
2. **추출**: 녹음 종료 → 오른쪽 위 점선 세 개 → 음성 기록 → 다운로드(텍스트 추출)
3. **요약**: ChatGPT(20분 이하) 또는 Claude(20분 초과)에 플러스(+) → 파일 업로드 → 결과물 지정 프롬프트 입력

**프롬프트 틀**: "회의 내용을 [글자 수]로 요약 + 초등학생도 이해하기 쉽게 [글자 수]로 설명 + 창의적인 아이디어 [글자 수] 제공"
- 상부 보고용 → "회의록으로 작성해 줘" 추가
- 발언자 특정 → "썸네일에 대해 말한 사람은 참석자 1이고 이름은 OOO야"처럼 맥락 보충
- 용도 예: "유튜브 대본으로 짜 줘", "사업 계획서로 작성할 거야, 참석자별 결론에 대해 이야기해 줘", "창의적인 아이디어를 3,000자로 제공해 주세요"

**원칙·주의**
- 단순 "요약해 줘"만 입력하면 AI가 알아듣기 어려움 → 반드시 용도·분량·형식 구체화
- 맥락(참석자 이름·역할·발언자 구분)을 추가할수록 품질 상승
- 부동산·변호사 등 전문 용어 많은 상담에도 "초등학생도 이해하기 쉽게 설명해 줘" 적용 가능
- 한국어 대화 중 영어 단어(SEO 등)도 정확 인식, 60~70대도 사용 가능한 단순 플로우
- 20분 초과 녹음을 ChatGPT에 넣으면 처리 어려움 → Claude 사용
- **법적 주의**: 본인이 참여한 회의·통화 녹음은 합법, 제3자 몰래 녹음은 불법

### 7. AI 트렌드 — 마케팅 채널로서의 AI

### 후킹·카피라이팅·영상광고 공식

#### 영상 광고 3단 구조 (출처: 나락 자청의 필살기 영상광고)
1. **후킹(감정 트리거)** — 타깃의 보편적 죄책감·고민을 첫 문장에서 건드려 몰입 유도 (맨 처음 1~2초가 가장 중요, 없으면 즉시 이탈)
2. **공감·문제 나열** — 타깃이 공통으로 겪는 고민을 구체적으로 설명해 '내 얘기네' 만들기
3. **해결·증거 제시** — 제품·서비스로 문제가 어떻게 해결됐는지 수치/사례로 입증
- 핵심 공식: **본능 분석 + 반박 제거** (타깃의 본능적 욕구·죄책감 분석 → 구매 망설임 반박 포인트를 미리 제거). 상세 페이지·랜딩 페이지·SNS 영상 광고·대인 관계에 모두 적용되는 범용 원칙.

#### 유튜브 조회수 4단계 초반 공식 (출처: 유튜브 조회수 3배 되는 4가지 공식)
영상 오프닝 10~20초 안에 모두 배치. 후킹이 맨 앞이어야 하며, 한 장면에서 4가지 동시 충족도 가능.

| 단계 | 이름 | 정의 |
|---|---|---|
| 1 | **후킹(Hooking)** | 시작 2초 안에 놀라운 장면·질문·궁금증으로 집중도 끌어올리기 |
| 2 | **가치 입증** | 화자가 이 말을 할 자격이 있음을 성과·경력·수치로 증명 |
| 3 | **라포르(친밀감)** | "나도 너와 똑같은 사람" (흙수저·실패 경험 등 공통분모) → 거리감 해소 |
| 4 | **도파민** | 새롭고 특이하고 신기한 것 제시 → 궁금증·놀라움 유발 |

- 가치 입증만 하고 라포르 생략 시 "너니까 가능한 거야"로 이탈. 라포르 필수.
- 썸네일에 가치 입증 요소를 박으면 본문에서 가치 입증 생략 가능(예: 썸네일에 "서울대 황농문 교수"). 단 후킹은 반드시 영상 내부에 있어야 함.
- 지식 채널은 전문 용어를 한두 개만 전략적으로 삽입 → 권위감·도파민. 너무 많으면 역효과.
- 영상 초반 웃음/유머 포인트 삽입 → "진지하지 않다"는 인식 → 시청 지속률 상승(라포르 도구).
- 우울·무기력 주제는 장기 검색 콘텐츠 → 공개 2~3년 후 조회수 복리 누적.
- **유튜브 콘텐츠 두 갈래**: (A) 4단계 공식형(지식·인터뷰, 일반인 가능) / (B) 순수 도파민형(몰카·브이로그·세계여행, 외모·창의력·특수환경 필요해 일반인 어려움).
- 후킹 예시 원문: "유튜브로 성공하는 것은 실력일까요, 운일까요? 165만 유튜버인 제가 무명 시절로 돌아간다면…" / "갑자기 꿈이 없는 사람은 개새끼예요" / "어떤 비난에 화가 난다는 건 다 내가 동의해 주는 거예요"

#### 숏폼 콘텐츠 4가지 공식 (출처: 빵원 릴스마케팅 자청의 4가지 공식)
1. **표본 이론** — 카피·썸네일이 반응할 잠재 독자 수를 최대한 넓혀라. "사이코패스 거르는 5가지 책"(50만 명) → "김경일 교수가 추천한 5가지 책"(전 국민) → 조회수 100배 차이.
2. **갈라치기** — 두 집단(A vs B)을 비교군으로 배치, 한쪽엔 손해 한쪽엔 이득. "MZ세대는 ○○하고 / 기성세대는 ○○한다"처럼 라임 맞춰 두 줄 대비. 뇌는 손해·이득 대비에 즉각 반응.
3. **궁금증 유발(Curiosity Loop)** — 결론을 빨리 보여주지 않고 5초 이상 '보여줄 듯 말 듯' 당겨 심리적 불편함 조성 → 해소 순간 도파민. (마스크 벗기 영상 6~7초 끌기 → 폭발)
4. **언매칭(패턴 파괴)** — 기본 패턴(상식)과 정반대 단어를 의도적으로 붙임. "재회하고 싶으면 매달리지 마세요"(기본) → "재회하고 싶으면 쌍욕을 하세요"(60만 뷰). 기본 패턴을 알아야 꼬을 수 있으므로 평소 플랫폼을 많이 보고 패턴을 DB화.

#### 릴스 초대박 3공식 (출처: 역행자 자청의 릴스 초대박내는 공식 3가지)
1. **피카소 이론** — 처음부터 창의 금지. 잘된 콘텐츠의 형식(포맷)을 먼저 모방하고 내 콘텐츠를 얹는다(내용 표절 ≠ 형식 차용).
2. **표본 이론** — 표본 넓은 예: GPT 거짓말(500만 명), 부자 되는 법, AI PPT, 중학생도 할 수 있는 ChatGPT / 좁은 예: 프롬프트 잘 만드는 법(1~2만 명), 미드저니 광고, LLM 개발.
3. **반응도 이론** — 알고리즘 핵심 지표 = 시청 지속 시간 + 공유 + 댓글 + 좋아요 + 저장 + DM. 콘텐츠 기획 단계부터 반응도를 의도적으로 설계(시청자가 특정 행동을 할 수밖에 없도록).
- 실증: 발리 릴스 1개 → 이틀 만에 240만 조회수, 팔로워 4,000명 / 저장 3.6만, 보내기 2.3만, 좋아요 1.2만, 댓글 6천.
- CTA 설계: 릴스에서 복붙 불가 점 이용 → "사용법은 캡션/댓글" 유도 → DM 응답으로 프롬프트 제공.
- 채널 초반에 지인·팔로워에게 공유/구독 요청 금지 → 관심 없는 팔로워는 알고리즘에 악영향.

#### 잘 터지는 릴스/쇼폼 18가지 공식 (출처: 자신있습니다. 온라인마케팅 2시간만)
1. 분석 어셈블(가장 중요, 첫 5초 음악·장면·문구·감정 쪼개기) 2. 숫자의 원칙 3. 특이한 장면 4. 유행 음악(1~2개월마다 교체) 5. 업매칭/언매칭(상반 개념 조합: 람보르기니 타던 사람이 자전거, 서울대 교수가 죽고 싶었다) 6. 박자 맞추기 7. 1초의 법칙 8. 잘못 봤나(어? 뭐야?) 9. 스토리텔링(기승전결·감정 곡선) 10. 감성 11. 논란거리 제공(댓글 싸움 유도, 고급) 12. 귀여움 13. 특이한 배경(판테온·콜로세움) 14. 표본 이론(첫 장면 표본 최대화) 15. 가치 깎기(가치 높은 사람의 실패·슬픔) 16. 꼬아주기(유행 보여주다 "절대 하지 마세요" 반전) 17. 팔로우 이유(재미 넘어 삶에 도움) 18. 박자/화면 전환.
- 쇼폼 제작 실무: 레퍼런스 전용 비즈니스 계정 생성 → 100만 조회 100~200개 수집 → 카피 + 영상 전개 분석. 한 장면 3초 이하. 인스타 데드존(상단 300px·하단 400px) 텍스트 금지. 썸네일 카피 15자 이내. "뭐뭐하지 말고 이거 하세요" 명령형 > 서술형.

#### 유튜브 역공학(주제 베끼기) 프레임워크 (출처: 5분만에 조회수 구독자 3배)
1. 내 채널 카테고리 키워드로 유튜브 검색 (영문법/영어 단어/영어 독해 각각)
2. 검색 필터 → 조회수순 / 길이 4~20분 / 전체 기간
3. 상위 콘텐츠의 썸네일·제목 패턴(숫자, 해결, 시간 단위)을 추출해 내 식으로 변형
- 핵심: "주제(포맷·공식)를 베끼는 것이지 영상 내용을 베끼는 게 아니다." 자기 머리로 주제 만들면 조회수 100, 역공학 적용 시 3~10배.
- 잘 통하는 제목 포맷: "영어 어순 15분이면 해결된다"(342만 회), "10분 투자로 형용사 부사 평생"(29만 회), "○○하지 마세요", "○○의 배신", "44점에서 91점 떡상", "30대 초반에 자동으로 월 1억씩 버는 방법". 시작은 강의를 그대로 녹화해 핵심 3~5분 클립 업로드부터.

#### 뷰트랩(ViewTrap) 활용 3단계 (출처: 자청이 뷰트랩 써서 조회수 70만)
1. **영상 트렌드 검색** — 키워드 입력 후 필터(기여도·성과도: 굿/그레이트) 적용 → 알고리즘 타는 영상 추출
2. **채널 분석** — 경쟁 채널 등록 → 굿·그레이트 + 쇼츠 제거 → 잘된 영상 30개 집중 분석
3. **카피 변형 적용** — 제목·썸네일 워딩을 그대로 따라하지 말고 반드시 변형
- 기여도(구독자 대비 폭발) vs 성과도(절대 조회수) 구분, 둘 다 굿·그레이트로 걸어야 진짜 인사이트. 노출 확률은 시의성 이슈 없으면 참고만.
- 제목 공식: "가난한 사람은 이렇게, 부자는 이렇게"(비포·에프터 대비, 62만 회), "돈도 백도 없으면 [X]만 하세요", "[숫자]만 하세요", "[권위자] + 주장"(자청이 아닌 "서울대 황농문 교수"가 말한다 형식). 썸네일: 비포·에프터(원 구분), "충격적인 [형용사]+[명사]", 순자산 숫자 노출.
- 같은 공식 반복 남발 금지 → 표본 박살. 완전 카피 금지(변형 필수). 악플 증가 = 잘되고 있다는 신호.

#### 숫자의 원칙 (출처: 5분만에 조회수, 압구정 의사들, 이제 직원 0명)
- 제목·썸네일·카피·프로필·셀링 포인트에 **반드시 구체적 숫자** 삽입.
  - 기간: 10분, 15분, 3주, 8분 만에, 20초 만에
  - 성과/점수: 44점→91점, 99% 좋아졌어요, 3배, 10배
  - 프로필/실적: "임플란트 9,754건", "맛집 1,000군데 방문", "한 달 300명 진료하는 의사", "70kg→43kg 감량한 [이름]"
- "잘한다"는 0점. "10년간 3,000명 지도, 비포·애프터 100장"처럼 반박 불가한 수치로 바꿔야 결제 발생.

#### 마인드 리딩 + 가치 입증 표준 블로그 글 구조 (출처: 이제 직원 0명으로 마케팅)
1. 마인드 리딩(공감) 도입부 — 독자 불안·고민 2가지를 먼저 맞혀 공감
2. 가치 입증 — 연수·판매량·비포애프터(숫자의 원칙)
3. 구체적 증거 제시 — 리뷰·사례·수치(반박 불가 형태)
4. CTA — 연락처·링크·사진 위치 안내(신뢰 형성 전 CTA 금지)
- 추상어 절대 금지("압도적", "뛰어난", "최고의") — 자청이 유일하게 화내는 지점.
- 방어기제 건드리는 도입부 금지("한 번만 오면 다른 매장 필요 없다"류 과장).
- 콘텐츠 생성 3대 입력값: 업체명 + 잡고 싶은 키워드 + 셀링 포인트(구체 수치 비교: "한우곰탕 32년, 타집 1~2일 숙성 vs 우리는 한 달 이상 숙성").

#### 기타 후킹/카피 전술
- **언매칭 사업 컨셉** (출처: 빵원 릴스): "영업도 하지 않고 전화번호도 없는 이상한 마케팅"(2018, 업계 영업전화 관행을 뒤집음).
- **궁금증 유도형 첫 줄** (출처: 쓰레드): 첫 줄에 숫자, 2~3줄로 궁금증만 유발 → 클릭 → 본문 펼침.
- **호기심 유발 문구 > 제품명 노출** (출처: 릴스 100만 왕초보): "1등 하면 생기는 일"(역행자 책 모르는 사람도 클릭, 22만 회). 책·제품명 전면에 내세운 평범한 홍보 문구는 이탈.
- **갈라치기 키워드 응용** (출처: 자청이 말하는): "예적금하지 말고 이거 하세요" → "재회할 때 매달리지 말고 이거 하세요"(업종 응용).
- **손실 위협 카피** (출처: 온라인마케팅 2시간): "안 하면 20% 가산세", "이거 모르고 등록하면 100만 원 날린다".
- **블로그 제목 비포/애프터** (출처: 온라인마케팅 2시간): BAD "스피닝하고 횡문근 융해증"(지식의 저주) → GOOD "원주 헬스 이거 모르고 등록하면 100만 원 날린다. 20년차 물리치료사…".
- **지역 소상공인 카피** (출처: 2024 최신 마케팅 7가지): "아산에서 가장 정직한 카센터를 찾고 계시죠?" / 이별 상담: "혹시 이별하셨나요?".

### 가격설정 전략

#### 가격을 높이는 3가지 기술 (출처: 매출 2배 순수익 5배 가격설정)
1. **친절함** — 상대가 궁금해할 것을 모두 격파(정보의 완전한 제공). 후기 게시판 / 성공사례 / 환불 정책 / 운영 철학 / 교육 과정 상세 / 사진 품질 / 위생·매장 사진 / 주소·방문 시간.
2. **줄세우기** — 마케팅으로 실제 초과 수요를 만든 뒤 가격을 점차 올린다(초기 저가 → 초과 수요 → 단계적 인상).
3. **불편해소(본질 강화)** — 고객 불편을 하나씩 제거. 과거 수료생에게 최신 자료 무료 제공, 하위권 무료 보충 수업, 5~10년 전 고객에게 칼럼 발송.

#### 순이익 레버리지 공식 (출처: 매출 2배 순수익 5배)
- 판매가 2.9만 원(원가 2.4만) → 순이익 5,000원 / 판매가 3.4만 원 → 순이익 1만 원.
- **가격 차이 15% = 순이익 차이 2배.** 가격 인상이 매출보다 순이익에 훨씬 큰 레버리지.

#### 망하지 않는 3단계 + 가격 인상 타이밍 (출처: 사장.대표는 ㅈㅇㅅ만, 사장과 망하는 사장)
1. **본질 업그레이드** — "내가 이 상황이면 무조건 산다" 확신 들 때까지 개선 ("이거 안 사면 바보" 수준).
2. **마케팅 병행** — 온라인 마케팅 꾸준히 학습·적용.
3. **초과수요 → 가격 인상** — 임의로 올리지 말고 과수요 발생 시에만 올린다.
- **1·1 원칙**: 하루 본질 1 업그레이드 + 하루 신규 고객 1명(마케팅). 본질 × 마케팅 = 성패.
- 자의식 제거: "내가 이만큼 고생했으니 이 가격" = 절대 금지. 망하는 원인 90%+가 자의식 개입.
- 가격 변천 실증(아트라상 이별상담): 1.5만 → 5만 → 10만 → 20만 → 22만 → 39만(최고 90만), 대기 한 달. 초기 대비 약 10배 인상.

#### 가격·협상 원칙 (출처: 월 1000 못 벌면 마케팅 교육)
- "비싸요"가 나오는 순간 = 상품 가치를 설득하지 못한 신호.
- 설득을 다 했는데도 비싸다 하면 받지 않는 게 낫다(클레임 가능성만 증가). 가격 깎기 시도 클라이언트 = 자격 없는 고객으로 분류·거절.
- 가치 제안 클로징: "너 이거 300만 원 들이면 1,000만 원 이상 확률적으로 무조건 가져가. 할 거야 말 거야?"
- 본질 강화 가능 품목(교육·소프트웨어·서비스업)을 선택할수록 가격 인상 여지 무한. 커피처럼 본질 업그레이드 한계 있는 품목은 불리.
- 전액환불제도 같은 리스크 리버설 장치 = 상품 확신의 시그널.

### 제안서·세일즈

#### 제안서의 본질 + 비즈니스 스케일 공식 (출처: 제안서 작성법만 알아도)
- 제안서 = 형식이 아니라 "상대 니즈 파악해 먼저 제안하는 모든 행위"(메모장에 짧게 써도 제안서).
- **본능 분석과 반박**: 받는 담당자 심리를 먼저 분석("복붙이냐 아니냐"로 사람 판단) → 반론 해소를 제안서 안에 넣는다.
- **제안서 3단 구조**: ① 공감/도입(짧게, 3분 내 읽힐 분량) ② 품질·신뢰 증명(사진·후기·재구매율) ③ 상대 회사 언급/마무리 진심.
- **장사 vs 비즈니스 공식**: 소수 개인 직접 판매 = 장사 / 타깃 1,000명+에게 제안서 시스템화 = 비즈니스. **계약률(%) × 모수(발송 수) = 매출.**
- 발굴 채널: 네이버 유통 카페("물건 찾습니다/판매처 찾습니다/제안 주세요"), 박람회 명함, 지식인·카페 Q&A 직접 답변, 대학 게시판.
- 실증: 한식 디저트 공방 → 카페에 12개 발송 → 4개 계약(33%) → 1,500만 원.
- AI 활용: 상대 회사 홈페이지 정보를 ChatGPT/Claude에 입력 → "세 줄 요약" → 개인화 문구 자동 생성(대량 발송도 개인화).
- 좋은 도입부: "21세기 북스 담당자님, 제가 [책]들을 읽고 이 출판사에 관심이 있었고, 이 출판사는 이런 강점이 있다고 생각합니다. 제 책은 이런 강점이 있어 핏이 맞을까 해서 제안드립니다."
- 마무리: "보셨을지 모르지만 이런 회사도 알게 되고 제안드려 기뻤습니다. 읽어 주셨다면 감사합니다."
- 자소서 응용: "귀사는 이런 사람을 뽑을 수밖에 없는 이유가 있고, 저는 이런 사람입니다."
- 금지: 복붙 제안서(0.1초 만에 들킴), 타깃 아닌 무작위 대량 발송, 제품 자랑만(상대 회사 조사+연결점 필수), 분량 과다(3분 초과), 증거 없는 주장.

#### 월소득 베팅 제안서 공식 (출처: 1000억 자산가들이랑 밥먹는법)
- **2가지 감정 트리거**: ① 영감(상대가 나로부터 돈 버는 아이디어 받을 느낌) ② 옛날 내 모습/동질감("내 젊었을 때 저랬는데" → 키워주고 싶은 본성).
- 공식: "저 월 OOO만 원 벌고 있습니다. 이번 달 번 돈 OOO만 원 드리겠습니다. 1시간만 만납시다."(예: 1,500만 원 → 현승원 3,000억 자산가 제안).
- 차별화는 구체적 숫자에서: "커피 한잔" vs "1,500만 원 드리겠습니다"가 만남 성사 여부를 가른다. 월소득 내거는 것 = 금액이 아니라 '판단력 있다'는 신호.
- 기댓값 베팅: 실제로 돈을 받지 않더라도 제안 자체가 특별함을 만들므로 제안만으로 성공.
- 실증(3천억 자산가): 수천 명 거절했으나 영상에서 배운 방식대로 맞춤 메일 보낸 1명만 만남 성사.
- 금지: "그냥 만나자/커피 한잔/맛있는 거 사드린다" 즉시 차단. 이미 알려진 수법 모방은 변형 필요.

#### 일잘러의 제안서: 비판 → 수치적 대안 (출처: 일 잘하는 사람 vs 일 못하는 사람, 제안서 작성법)
- "비판은 누구나 한다. 일 잘하는 사람은 비판에 수치적 근거 + 대략적 결과값을 붙인 대안을 만든다."
- 구조: ① 문제(비판) ② 대안 제시 ③ 비용 계산 ④ 기대 효과 수치화(% 향상·손실 절감액) ⑤ 근거 출처(책·뇌과학·데이터) ⑥ 부작용/리스크 언급(신뢰도↑) ⑦ 결론(장기 ROI).
- 수치는 완벽할 필요 없음 — 10~20% 예상치 대략 적용으로 충분.
- 실증(낮잠 공간 제안서): 100평 월세 1,000만 → 30%(낮잠방) 300만 비용. 100명 업무 향상도 110%, 번아웃 10% 감소 시 월 100만 절감 → 5개월 내 3,000만 수익.
- 안티패턴: "회사는 복지에 관심 없는 것 같아", "왜 ○○ 안 하지?" 같은 수치 없는 감정적 비판 = 능력 없는 사람의 특징.

### 마케팅 vs 브랜딩·퍼스널브랜딩

#### 브랜딩 vs 마케팅 정의 (출처: 마케팅 vs 브랜딩 7분 총정리)
- **브랜딩** = 이미지 구축 → 장기적 가치 / **마케팅** = 판매 촉진 → 단기 구매 유도.
- 고등학교 연애 비유: 브랜딩 = 1년간 공부·운동·배려를 학교 전체에 보여주는 것 / 마케팅 = "나 솔로야, 누가 괜찮아"라고 말하는 행위.
- **브랜딩 없이 마케팅만 하면 결국 망한다** → 필연적 가격 경쟁 → 도태.
- **브랜딩 선순환 구조**: 가성비 좋은 가격 + 기본 본질 → 구매 만족 → 재구매(20~30회) → 입소문 → 회전율↑ → 신선도·만족도↑ → 브랜딩 강화 반복.
  - 컴포즈 커피: 2,000원에 건당 500원 남기지만 재구매·입소문 합치면 건당 2~3만 원 구조 → 3,000억 매각.
- 브랜딩 = 약속의 일관성(매체·제품·행동에서 동일 메시지 반복). 백종원 솔루션 공식: **기본 본질 + 싸게 팔기**.
- 제품 하나가 브랜드 전체를 해칠 수 있다 → 신제품·콘텐츠 출시 전 "브랜딩에 해가 되는가" 먼저 검토(대충 만들어 단기 20억 벌면 나머지 브랜드 가치 전부 하락).
- 본질이 강하면 브랜딩 위기 극복(대한항공 땅콩회항, 자청 사이버 레카 사례).
- 대기업 계열사처럼 하나의 브랜드에 너무 많은 카테고리를 묶으면 브랜딩 분산(삼성 vs 애플 일관성).

#### 퍼스널 브랜딩의 함정 (출처: 돈버는법 강의가 사기일 수밖에)
- 실력 없는 상태의 퍼스널 브랜딩(강의·지식 판매) = 사기. "퍼스널 브랜딩 = 문제 해결사가 아니라 병신이 되겠다는 것."
- **사업 = 문제 해결력**(문제를 발견하고 하나씩 해결하는 반복 과정). 구글·아이폰처럼 수천 가지 문제 해결.
- **퍼스널 브랜딩 시작 기준선**: 순수익 월 3,000만 원 이상 달성 후에만 지식 공유 고려.
- 자기 점검 지표: 강의·콘텐츠 소득이 본업 소득보다 높아지면 경고 신호(실력 아닌 '팬덤 자위'에 의존).
- 구조적 악순환: 실력 없이 강의 팔면 수강생도 똑같이 강의 판매로만 돈 버는 방법을 배운다 → 사업 실력 0.
- 순서: 성취(월 순수익 3,000만+) → 결과 공유(퍼스널 브랜딩). 역순은 사기.

#### 스토리텔링 = 입증 + 구매욕 (출처: 무료 마케팅인데 매출 2배)
- 사장의 창업 스토리·차별화 이유를 공개하면 무의식적으로 신뢰·애정 형성 → 구매 전환율 상승(스토리 접하면 그 사람에게 애정 생김).
- 셀프 사진관 캡션 예시: "사진 찍는 게 항상 불합리하다고 생각했다. 알바 며칠 치 월급이 10분 사진에… 셀프 사진관을 하면 가격은 20~30% 낮아지면서 이 모든 문제 해결…"
- 모든 마케팅은 처음 온 사람 기준 — 팔로워가 같은 캡션 반복으로 봐도 새 유입자가 타겟이므로 스토리텔링 유지.

|---|---|---|
| 지역 + 정보 비대칭 | 카센터, 지역 피부과/인테리어, 심리상담, 세무사, 학원 | **네이버 블로그** |
| 지역 + 이미지 | 지방 맛집/카페/북카페/마사지샵, 미용실, 사진관 | **네이버 지도(플레이스) + 블로그** |
| 전국 + 정보 비대칭 | 마케팅 회사, 이별 상담, 출판사 | **유튜브 롱폼 + 릴스** |
| 전국 + 이미지 | 호텔, 전시회, 캠핑장, 온라인쇼핑몰, 로고 디자인 | **릴스·쇼츠·숏폼 + 퍼포먼스 마케팅** |

- 규칙: 지역 → 검색 기반(블로그·지도) / 전국 → 불특정 다수 노출(릴스·유튜브) / 정보 비대칭 강하면 블로그 병행.
- 경계 사례(강남·청담 등 유동인구 많은 상권) = 지역과 전국 중간 → 블로그 + 릴스 병행.
- 핵심 규칙: 주력 하나에 집중, 나머지는 "잔잔발리(보조)" 수준. "하나만 해라"가 아니라 "하나에 집중".
- 본질 강화 가능성이 사업 선택 기준 — 본질 강화 불가 아이템(탕후루·무인카페·중국 위탁판매)은 마케팅으로 커버 불가, 가격 경쟁 패배.

#### 정보 비대칭 해소 모델 (출처: 2024 최신 마케팅 7가지, 4사분면)
- 소비자가 품질을 알기 어려운 업종(중고차·변호사·의사·마케팅·심리상담)일수록 전문 지식 콘텐츠 공개 → 신뢰 획득 → 구매·의뢰 전환.
- 브랜디드 유튜브 채널은 직접 광고 없이 전환 가능(전문성만으로 신뢰 형성).

#### 표본 이론 + 피카소 이론 + 깔때기 이론 + 작은 행동 유발 (출처: 베스트셀러 작가 4가지)
1. **표본 이론** — 도달 가능한 수요자 모수 최대화. 주제 좁을수록 표본 쪼개져 결과↓. '영양사 취업법'(2,300명) vs '부동산'(1,000만 명) vs '경제적 자유'(3,000만 명). 중식 요리사가 '중식 책' 내면 10만 부 가능자가 1만 부로 축소.
2. **피카소 이론** — 기존 시대 기초·유행을 마스터한 뒤 자신만의 것을 완성. 내 분야 20년 베스트셀러 2~3권의 형식 분석·흡수. 『부자아빠 가난한아빠』·『부의 추월차선』 공통점(비교군 개념) → 역행자 vs 순행자 구도 + 첫 페이지 스토리텔링. 형식 차용 ≠ 내용 표절.
3. **깔때기 이론** — 난이도 흐름 쉬움→중간→어려움. 앞장은 전 국민 공감(표본 넓게), 뒷장은 심화(무자본 창업) → 일반 독자 "쉽고 재밌다" + 지식인 "심도 있다" 둘 다 충족.
4. **작은 행동 유발** — 독서 도중 '작고 쉬운 실행'을 명령("지금 책 덮고 핸드폰 끄고 5분 산책"). 독자가 실제 행동 변화 경험 → "좋은 책" 평가 → 바이럴. 큰 행동 강요하면 무시.
- 핵심: 자의식 해체. "유튜버가 남의 유튜브 안 본다 = 멍청하다." 이름값 없을 땐 피카소·표본 이론 필수, 이름값 생긴 후 자기 방식 허용.

#### U-I-S 아이디어 생성법 (출처: U-I-S 3단계 자청의 아이디어 생성법)

| 단계 | 원어 | 핵심 질문 |
|---|---|---|
| U | Uncomfortable(불편함) | "나/타인이 지금 무엇에 불편한가?" |
| I | Idea(아이디어) | "그 불편함을 어떻게 해결하면 되지?" |
| S | Specimen(표본) | "이 해결책을 원하는 사람이 얼마나 많은가?" |

- 사업·책·유튜브·강의·콘텐츠 모두 동일 적용.
- 실행: ① 불편함 즉시 메모 ② 최소 단위 해결책(앱·직원·투자 말고 카카오 오픈채팅·전단지·블로그 글 하나) ③ 표본 추산(낮으면 폐기/타겟 교체) ④ 작게 검증 ⑤ 신뢰 불편함 해소(영상통화·신분증 등) ⑥ 표본 확대(타겟 넓히기).
- 핵심 인사이트: 불편함(고통) 클수록 지불 의향↑(장례·이별 상담은 표본 좁아도 수익 큼). 표본 확대 = 수익 확대(두더지 퇴치기: 한국 표본 소 → 미국 농부로 교체 → 대박). 포지셔닝(빈자리)이 핵심. 신뢰도 자체가 불편함의 일부. "재미만 주는 콘텐츠"는 돈 벌기 어렵다.
- 실증: 라오스 기차 예매 대행(오픈채팅, 건당 5천~1만 원), 역행자(기존 명저가 외국·구버전 → 한국·최신·현직 고수익자), 몰리의 방(스마트폰 잠금 앱), 아트라상(이별 상담), 이상한 마케팅(사업 경험 있는 사람이 운영하는 대행).

#### 모텔 이론 (신뢰 단계 구축) (출처: 자청의 필살공식, 웹사이트 제작 문의 300번)
- 처음 만남에서 곧바로 상품을 꺼내는 것 = 처음 본 사람에게 바로 모텔 가자는 것 → 절대 금지. 단계별로 능력·전문성·신뢰를 쌓은 뒤 자연스럽게 상품 인지.
- 채널별 깊이 조절: 숏폼·릴스는 가볍고 대중적, 쓰레드는 중간, 유튜브 롱폼은 깊은 내용. 인스타 광고는 예외적으로 직접 상품 알리되 숫자 원칙·공감·반박 격파 삽입.

#### 의심 격파 프레임워크 (홈페이지 전환율) (출처: 웹사이트 제작 문의 300번)
1. **고객 의심 리스트 작성** — 처음 방문자가 품을 모든 의심 나열(이별 상담: 가격? 실력? 사기? / 마케팅 회사: 등쳐먹나? 성과? 사무실 큰가? / 프랜차이즈: 먹튀?).
2. **의심별 증거 제시** — 사진·후기·수치·문구.
3. **조급함 제거** — 결제 유도보다 고객이 원하는 것을 먼저.
- 추상어 금지 → 구체성 최대화("가정의 평화" → "11년간 566개의 가정", 『스틱』의 구체성 원칙). 간접 증명("우리가 짱이다" 대신 후기·사진·수치). 첫 화면에 계좌번호+문의하기 배치 = 즉각 이탈.
- 거만함의 역설(아트라상): "후기·무료 칼럼 안 읽고 1위라는 이유만으로 신청하면 거절합니다" → 무료 강조로 결제 압박 제거 + 역설적 신뢰.

#### '먼저 이득을 줘라' 6채널 전략 (출처: 자청의 필살공식)
| 채널 | 핵심 원칙 |
|---|---|
| 릴스 | 대중적·가벼운 정보 + 불편 해소(대중성×유익함) |
| 쓰레드 | 정보성만, 상품 언급 금지(똑똑한 층 → 조금 깊게) |
| 유튜브 롱폼 | 깊은 전문 정보로 실력 증명, 런칭 6개월 전부터 선작업 |
| 인스타 광고 | 모텔 이론 예외(직접 상품 알림), 숫자 원칙+공감+반박 격파 |
| 블로그 | 팬 기반 알림, 판매보다 정보성 |
| 광고 맡기기 | 인플루언서 체험단·협찬 위탁 |
- 본질 강화: 30만 원에 60만 원 가치 제공 → 만족도 차이만큼 재구매·브랜드 강화(차액은 손해가 아닌 미래 브랜드 자산).
- 입소문은 전략이 아닌 결과: 본질 높으면 자동 발생, 런칭 초기 할인→충성 고객 빠른 소화→자연 확산.
- 누구나 '1등'은 있다: 분야를 충분히 좁히면 1등 포지션 가능("3개월 만에 정보 채널 수익 상승률 1위"). 없으면 GPT로 포지셔닝 컨설팅.
- 인스타 광고 스크립트 흐름: 타깃 공감 후킹 → 고통 공감 → 라포르 → 숫자+포지셔닝("하루 평균 308개 팔리는") → 반박 선제 제시 → 반박 격파 → 경계심 해제("지금 안 들으셔도 돼요, 무료 강의도 열려 있어요") → 간접 클로징.

### AI 프롬프트 6공식 (전략 활용)

#### AI에게 일 시키는 6가지 프롬프트 공식 (출처: 자신있습니다 온라인마케팅 2시간)
1. **역할 부여** — "너는 20년차 세계 최고의 블로그 마케팅 전문가야"
2. **목표 설정** — 업무 목적·배경 설명("작성한 글을 읽은 잠재 고객이 매장에 방문하고 싶게")
3. **형식 지정** — 글자수·도입부 조건·말투 구체화
4. **예시 제공** — Good/Bad 예시 첨부 → 퀄리티 급상승
5. **깊게 고민(Chain of Thought)** — "천천히 생각하고 답변해 줘"
6. **동기 부여** — "회사 매출을 결정하는 중요한 일, 잘하면 100억 줄게"(유료 프로 버전은 생략 가능)
- 도구별 용도: ChatGPT(기획·제안서·딥리서치) / Claude(블로그 글·CS 문구) / Perplexity(시장 조사).
- '황당하고 창의적인 전략 2개' 요청 공식(출처: 상위 1% 일잘러 ChatGPT): 일반 전략 외에 황당한 전략도 요청 → 10개 중 1~2개 얻어걸려 디벨롭. 질문의 질 = 답변의 질(사업명+서비스+운영방식+가격+매출+순이익 모두 입력).

### 잘되는/망하는 사업 구별법

#### 사업 판별 5대 조건 + 핵심 이분법 (출처: 이 구별법으로 평생 우려먹으세요)
잘되는 사업이 갖출수록 좋은 조건: ① 재구매율 ② **본질 강화 가능(가장 핵심)** ③ 원가·재고·유통 비용 적음 ④ 확장성 높음 ⑤ 객단가 높음.
- **핵심 이분법**: 사업의 90%+는 "본질 강화 가능 → 잘될 사업" vs "규모의 경제 의존 → 자본 없으면 망할 사업"으로 판별.
- 본질 강화 = "상품을 내 손으로 계속 개발·업그레이드할 수 있느냐" + 본질이 내 회사에 귀속되어야 함(외부 강사·공급자 귀속 시 취약 — 클래스101 vs 메가스터디).
- 생존 경로: 본질 강화 가능 → 일반 사업 OK / 본질 강화 불가 + 규모의 경제 가능 → 자본 있을 때만(쿠팡: 10년 적자 후 1위) / 본질 강화 불가 + 규모의 경제 불가 → 하지 말 것.
- "돈을 많이 버는가"만 보지 말고 "성장성·확장성 있는가"도 봐야(통신사 LG U+·KT: 수익 좋아도 국내 한정 → 주가 정체).

#### 피해야 할 2대 필터 (출처: 사업은 이 두가지만 피하면)
1. **단계(스텝) 수** — 돈 버는 데 필요한 단계가 적을수록 유리. 이상형: "나 상품 있어, 살래?"→"사!"→끝(1~2단계). 플랫폼(5단계+)·MCN·기획사는 초보에게 너무 어렵다.
2. **본질 강화 가능성** — 본질이 '싸게 파는 것'이면 경쟁사 진입 시 마진 붕괴(스마트스토어). 본질이 '실력/콘텐츠/품질'이면 진입장벽↑.
- MCN·기획사 = 사람 베이스 + 다단계 이중 리스크(소속자 자아 강해 이탈·소송). 카페로 돈 버는 법은 프랜차이즈(규모의 경제)뿐.
- 초보 때 스마트스토어·블로그·유튜브는 '레벨업 도구'로 유효, 주력 수익원으로는 한계.
- "뽕에 취해서" 멋있어 보여 진입 금지. 한 방에 월 3,000만 원 노리지 말고 50만→100만→200만 단계적 성장.

#### 사업 판정 공식 (AI 수식화) (출처: 2026년 망하는 업종 5가지)
- 분자(클수록 유리): 본질 강화 가능성 + 재구매율 + 확장성 / 분모(클수록 불리): 원가 + 재고 + 유통 비용 + 경쟁 강도.
- 판정: 1.2 이하 → 절대 하지 마라 / 0.9 → 갈아넣으면 버티나 비추 / 2.0 이상 → 매우 유리 / 5.0 만점 가까울수록 최상급.

| 업종/사업 | 판정 점수 |
|---|---|
| 두정쿠(도넛) | 0.44 |
| 무인편의점·무인카페·킥보드 | 1.2 이하 |
| 저가한우 프랜차이즈 | 1.2 |
| 카페 창업 | 0.81 |
| 클래스101 | 1.4 |
| 메가스터디 | 2.9(매년 신규 수험생 = 구조적 재구매) |
| 자청 사업 포트폴리오 | ~5점 만점 |

- **절대 가치 이론**: 정보량 많아질수록 소비자는 절대 가치(건강·실용·진짜 성능)로 수렴. 환상으로 파는 시대 끝(빙그레 희망퇴직, 페라리·벤틀리 주가 하락). 라라스(저당 아이스크림)는 절대 가치로 성공. 참고서 『절대 가치』.
- 본질 강화 무한 = 테슬라·AI 기업(전 세계 천재 수천 명이 본질 강화). 책 독서 = 자기 본질 강화(사고력↑ → 본질 강화 아이디어).

#### 망하는 업종 (창업 금지 리스트) (출처: 2026년 망하는 업종, 4사분면, 사업은 두가지만)
- 두정쿠·카스테라·탕후루류 유행 디저트(절대 가치 부재, 유행 지나면 소멸, 초기 진입자만 수익).
- 무인편의점·무인카페(본질 차별화 불가).
- 킥보드(국가 규제·여론 악재 + 1등만 생존, 현재도 적자).
- 저가한우 프랜차이즈(본사만 이익).
- 중국 소싱 단순 유통(스마트스토어·배드민턴 라켓 OEM, 본질 강화 불가 → 가격 경쟁).
- 셀프 사진관·킥보드 등 본질 동일화 레드오션 후발 진입.
- 클래스101식 플랫폼(본질이 외부 강사 귀속 → 이탈 리스크).
- 브이로그·웃긴 유튜브(콘텐츠 생산 한계).
- 디저트 전문점(파티시에가 릴스까지 직접 해도 월 순이익 5,000만 원 상한).
- 단일 카페·세제 등 범용 소비재 인터넷 판매.

#### 사업가 3분류 (출처: 스타트업 vs 경제적자유 vs 장사)
| 분류 | 심리적 뿌리 | 특징 | 한계 |
|---|---|---|---|
| 스타트업 | 배가 따뜻함(금수저) | 멀리 봄, 차별화·테크 기반, 확장성 무한 | 성공 난이도 극도로 높음 |
| 경제적 자유 추구자 | 어릴 때 극빈·결핍의식 | 눈앞 현금(월 1천~3천만) 집착, 단기 시야 | 단기 집착 |
| 장사(인력 기반) | — | 소규모 최적화(모텔·고깃집 순수익 2천) | 인력 의존 확장성 한계 |

- 스타트업 3특징: 배가 따뜻함 + 멀리 봄 + 테크 기반(확장성 무한). 스타트업은 세상의 비효율 제거로 돈 번다(야놀자·배민·카카오).
- 결핍의식 심리: 어릴 때 돈 공포 → 평생 돈 집착. 유전자(지능) 좋음 + 결핍의식 = 성공 조합.
- 테크 기반 아니어도 콘텐츠는 예외(광고 한 편으로 천억 가능). 플랫폼보다 제품 먼저(검증 후 플랫폼).

#### 잘되는 vs 망하는 사장 4가지 차이 (출처: 돈 버는 사장과 망하는 사장)
1. **방어기제 없음** — 핑계·변명·합리화 없이 "내가 게으르구나, 해야겠다".
2. **신문물 즉시 수용** — 사업/마케팅 = 전쟁(전차→전투기→스텔스→드론, 최신 무기 쥔 쪽이 승리). 새 채널·AI는 "일단 10분만".
3. **꾸준함** — 실패 후에도 남탓 없이 지속.
4. **좋은 집단(스터디·모임)** — 경쟁심·피드백·선순환.
- 실증: 70대 꽃 농장 운영자 숏폼으로 구독자 1만+, 60~70대 대표가 ChatGPT·GPTs·Claude 활용.

#### 본질 + 입소문 원칙 (출처: 사기꾼 마케팅 업체, 월 1000 못 벌면)
- "대박 상품 = 본질이 좋아야 + 마케팅 가미 → 입소문 자동." 역행자: 출간 직전 마케팅 집중 → 이후 거의 안 해도 2개월 10만 부, 종합 1위 5주.
- 50만~100만 유튜버가 책 내도 종합 1위는 하루이틀 반짝 후 하락(본질의 힘 vs 마케팅).

### 업종 전략 (병원 등)

#### 병원 마케팅 6가지 전략 패키지 (출처: 병원마케팅 6가지 전략)
1. **브랜드 블로그** — 원장 개인 블로그를 신뢰도 제고 전문 글로 육성(체험단·기자단 방식 X).
2. **원장 인터뷰 + 원내 사진** — 고객 심리 파악해 원하는 사진 기획, 원내 TV 배치.
3. **네이버 지도 상위 노출** — 합법적 5위 이내(불법 프로그램·허위 트래픽 금지).
4. **SNS 스폰서 광고(인스타)** — 병원 반경 1km, 나이·관심사 타겟.
5. **내부 CS 강화** — 미스터리 쇼퍼로 암행 점검, 예약 확인 문자로 노쇼 50%→10%.
6. **차별화 전략** — 원장 학력·전공·위치 강점 발굴(없으면 대행사가 설계).
- 공식: 매출 = 신환 유입(블로그+지도+SNS) × 전환율(CS+내부서비스) × 재방문(포지셔닝+신뢰).
- KPI는 노출량이 아니라 **매출**(상위 노출 수치·그래프는 가짜 지표). "안 되면 전액 환불."
- 블로그 구성 순서: 상대방이 궁금한 것(비포애프터·후기·경력) 먼저 → 전문성. 팔짱 낀 경력증 나열형 = 신뢰도 붕괴.

#### 병원 유형별 마케팅 우선순위 (출처: 압구정 의사들 2025년 병원마케팅)
| 병원 유형 | 1순위 | 2순위 | 3순위 |
|---|---|---|---|
| 지역 소형(안산·창원·정읍) | 블로그 직접 운영 | 네이버 지도 | 오프라인(현수막) |
| 강남 대형 | 유튜브 롱폼 | 블로그 대행 | 오프라인 버스 광고 |
| 초대형 확장 | DB 마케팅(선택) | 플랫폼 마케팅 | 전방위 믹스 |

- 블로그 제목: 진료 중 환자 질문을 그대로 활용("남양주 테니스 엘보 몇 개월 쉬어야 되나요"). 20분 작성·사진 2~3장.
- 유튜브 롱폼: 조회수 1,000~5,000의 표본 낮은 전문 영상 목표(대중성 높으면 의료법 신고 위험). 눈밑지방·코필러 등 특정 시술 집중. 한 번 올리면 3~6년 마케팅 효과. 롱폼 = 진상 필터링(다 보고 온 환자 = 신뢰·이해 완료).
- 썸네일 공략 심리(통증/시술): 빠르게 + 쉽게 + 집에서 + 병원 안 가도 됨 중 1개+ 삽입("누워서 3분씩 했더니 99% 좋아졌어요", "8분 만에 허리통증 이별").
- **DB 마케팅 비추**: 2024~2025 레드오션. 과거 만 원 투자→50~100만 환자 / 현재 50만 투자→70~100만(손실 구조). 중소 병원 남용 금지.
- 타겟 연령: 7~80대면 버스·현수막 / 2~30대면 온라인 압도.
- 실증: 압구정 탈모 병원 → 블로그·롱폼·쇼츠 병행 → 개원 2년 이내 월 매출 1.5억~2억.

### 대행사 고르는 법·마케팅 교육의 실체

#### 대행사 고르는 5단계 (출처: 사기꾼 마케팅 업체 거르고)
1. 영업전화하는 곳은 피한다(본질 안 좋을 신호, 좋은 서비스는 입소문으로 온다).
2. 책 1권 이상/관련 글 읽고 사전 공부 후 미팅.
3. 반드시 미팅 — 3~4군데 비교.
4. 미팅은 상대 회사에서(규모·환경 직접 확인, 원룸·오피스텔 1~2인인지).
5. 포트폴리오 확인 — 나와 비슷한 업종 사례.
- 핵심 질문: **"제 상황에 맞는 마케팅이 무엇인가요?"**(통찰력·기술 수준 전부 파악).
- 대행사 vs 실행사: 대행사(10인+, 매출 검증) / 실행사(1~2인이어도 특화 기술 있으면 신뢰, 카페 침투 프로그램 등).
- 대표가 대행사를 컨트롤할 수 있을 만큼 알아야 사기 안 당한다. 초반엔 대행 없이 대표가 직접.

#### 대행사가 사기를 칠 수밖에 없는 4가지 구조 (출처: 온라인 마케팅 대행사의 4가지 문제점)
1. **사업 무경험** — 본인 사업 못 해봤는데 남의 것 성공 어렵다.
2. **매체사 수수료 구조** — 받은 금액 90%가 네이버·인스타·체험단으로 나감(3,000만 수주→300만 남음→직원 월급 주면 적자, 보통 100만 수주→실수익 10만).
3. **인력 이탈** — 기술 배운 신입은 배우자마자 독립(대표 1명+신입 반복 순환).
4. **고객층 불일치** — 잘하는 대행사는 월 500만~3,000만 쓰는 고객만 받는데, 그런 고객은 이미 잘해서 필요 없음. 월 10만~100만 쓰는 자영업자는 안 받음 → '못하는 대행사'만 시장에 남음.
- "포스팅 많이 해드릴게요" = 본질 강화 무관한 실행 제안 → 거르기.

#### 마케팅 대행 vs 직접학습 기준선 (출처: 월 1000 못 벌면 마케팅 교육)
- 순수익 월 1,000만 미만 → 대행 금지, 교육·공부 단계(대표가 마케팅 고수가 먼저).
- 월 2,000~3,000만 달성 → 그때 대행 위탁(대표가 직접 하는 게 비효율 되는 전환점).
- 대행 위탁 전 대표 본인이 인스타·유튜브 등 주요 채널을 알고 있어야 관리·소통이 편함.

#### 고가 컨설팅의 실체 (출처: 초고가 사업컨설팅 마케팅교육의 진짜 실체)
- 천만 원+ 고가 컨설팅 대부분 진짜 사업가가 아닌 사람이 운영, 구조적으로 사기 경계.
- 판별: 판매자가 '사업으로 부자' vs '교육·컨설팅 판매로 부자' 구분(후자는 자기 방식만 가르침).
- **무조건 환불 제도 없으면 거르기**(본인도 효용에 자신 없다는 증거).
- 진짜 가치 검증: 성과 보장 시 '성과 비례 수수료'가 진짜(자신 있으면 무한 반복 가능, 자신 없으니 고정 선불).
- 레벨 구조: 레벨 3~4가 레벨 1을 가르치는 건 수십만 원 이내면 OK / 레벨 낮은 사람이 수천만 원 받으면 문제.
- 사업가는 컨설팅 안 함(김봉진처럼 사업에 집중 = 수천억). 신도·군단 구조, 조작된 성공 후기 경계.

#### "월 천만 원 누구나 가능"이 헛소리인 이유 (출처: 월1000 역대급 개소리)
- **회사 환원 논리**: 누구나 가능하면 직원 1,000명 교육+반반 쉐어 = 월 50억 → 안 하는 건 본인도 불가능함을 안다는 반증.
- **헬스 비유**: 재능자 vs 일반인 — 일반인도 훈련으로 40~50kg→80kg 가능(배움이 무의미한 게 아니라 '누구나 정상급'이 거짓말).
- **타이탄의 도구 전략**: 본질적 직업 1개 고정 + 독서·학습으로 지능 지속 상승 → 역량 복리 축적.
- 비즈니스 지식은 상위 50~30%만 들어도 먹고사는 데 지장 없는 분야(엔터처럼 최상위만 버는 게 아님). 논리력 필터: "진짜면 왜 그 방식으로 회사를 안 만드는가?"
- 양질 콘텐츠(역행자·오프라인 강의)는 선별 활용 — 교육 거부도 오류.

#### 마케팅 공부 5단계 (출처: 마케팅 공부법 5가지)
1. 서점에서 마케팅 책 많이 훑고 읽기(초기엔 실용서, 세스 고딘류 철학서는 후순위)
2. 오프라인 수업 수강(정체성 전환 효과 + 비용·대면 감시로 실행력↑)
3. 마케팅 툴 직접 실행(인스타 스폰서 등 결제까지 경험)
4. 온라인 강의 + 환경 설정(① 금전적 피해: 지인에게 돈 걸기 ② 사회적 평판: 스터디 완강 인증)
5. 유튜브 영상 → 블로그 필기 → 완성 문장 정리 → 장기 기억화.
- 마케팅 실력 = 한 순간이 아닌 근성장(수련). 주변 비전문가에게 묻는 건 "투수 폼 한 번 배우고 120km 던지려는 것". 무료 콘텐츠를 "돈 벌려는 거잖아"로 거부 = 팔 잘린 채 치료 거부.

### 마케팅 트렌드 큰 그림

#### 마케팅 트렌드 자가 계산 공식 (출처: 한 달에 1억 이상 벌면, 자신있습니다 2시간, 2024 최신)
- **핵심 원리**: "인간이 하루에 일어나서 가장 많이 접촉하는 시간·공간 = 마케팅 전쟁터." 이 공식으로 26·27·28년 트렌드 자가 예측 가능.
- 연대기: 1950s 신문·간판·입소문 → 1990s TV → 2006 네이버(배너 1시간 3천만 원) → 2012 스마트폰 → 2019~20 유튜브 롱폼 → 2024 숏폼(릴스·쇼츠·틱톡) → 2025 AI 에이전트.
- 매체 시간 변화: 틱톡/릴스 2019.8 (611억 분) → 2024.8 (약 2배). 인스타 1.8억→3.2억 분. 네이버·카카오톡 감소.

#### 2025 마케팅 채널 7단계 (출처: 한 달에 1억 이상 벌면)
1. AI 에이전트 2. 숏폼(릴스·틱톡·쇼츠) 3. 유튜브 롱폼 4. 네이버 블로그 5. 지도 마케팅 6. 쓰레드 7. AI 마케팅(콘텐츠·광고).
- **모든 채널에 조금씩 깔아두기**: 채널당 하루 1명 × 6채널 = 하루 6명 신규 고객.
- **구독자 성장 공식**: 6개 채널을 순서대로 하나씩 적용할 때마다 월 순수익 +1천만 원 → 1천만→1억(4년).
- 네이버 블로그: 파괴력 과거 100→현재 30~40이지만 기본값(지도 상위 노출 점수 + AI 검색 추천 점수 동시 확보).
- **AI 검색 노출 전략**: 웹·블로그에 질 좋은 글 + 업장 언급 축적 → AI(GPT·퍼플렉시티)가 검색 추천. 실증: "재회상담 1위" 검색 → 아트라상 1등(웹 트래픽↑+언급량↑+업력 긺).
- 정보비대칭 업종(의료)에는 롱폼 필수(실력 입증→브랜딩→전환).

#### 마케팅 연대기 + 채널별 성향 (출처: 2024 최신 마케팅 7가지)
- 2018년 이후 가장 효율 좋은 마케팅 = 인스타그램(릴스).
- 채널별 성향: 릴스(지적 수준 높은 층, 정보성) > 쇼츠(좀 더 가벼운 층) > 틱톡(10대·글로벌·재미).
- 유튜브 2방식: 퍼포먼스형(광고비로 강제 노출) vs 브랜디드형(전문 지식으로 신뢰→자연 전환, 자청 선호).
- 본질이 마케팅보다 우선. 콘텐츠 검증 후 광고(유기적 반응 테스트→검증 소재만 퍼포먼스). 감으로 만든 광고는 ROAS 안 나옴(초사오글쓰기: 하루 100만 광고→1,800만 매출, ROAS 1800%).

#### 쓰레드 마케팅 (출처: 10년하던 블로그 버렸습니다, 자신있습니다 2시간)
- 상위 5~30% 모수가 모이는 공간(글이라는 진화된 행위 = 상대적 고지능자). 글 2~3분 작성, 릴스 대비 공수 1/10.
- **반응도 공식(모든 플랫폼 공통)** = 체류시간 + 클릭 + 공유 + 리포스트 + 댓글 + 구독/팔로우. 쓰레드 핵심 지표 = 리포스트 + 공유하기.
- 5가지 원칙: ① 150자 이하·한 줄 20자 이하 ② 사진·영상 필수(체류↑) ③ 첫 문장 표본 넓게+숫자+강조 ④ 댓글 유도 ⑤ 리포스트·공유 10개 이상(알고리즘 선택 기준). 링크 삽입 시 조회수 반토막~1/3.
- 레퍼런스 분석 루틴: 리포스트+공유 10개 이상 글만 30분 저장 → 첫 줄/마지막 줄/흐름 분석 → 형식 차용.
- 프로필: 구체 숫자/실적("임플란트 9,754건"). 추상 표현("열심히 사는 사람") 금지.
- 스아이(SI)/품마(상호 댓글·좋아요·팔로우 교환) 절대 금지 → 반응 없는 계정으로 인식 → 노출 차단.

#### 카카오 숏폼 초기 진입 (출처: 카카오 숏폼 시작해야 되는 이유)
- **플랫폼 초기 진입 법칙**: 공급 < 수요 시점 진입 시 동일 실력으로도 조회수·수익 훨씬 큼(낮은 공급 × 높은 수요 = 초기 진입자 독식).
- 플랫폼 논란 = 진입 기회(논란 = 공급 감소 시그널). 기대값 사고(성공 불확실해도 확률×보상으로 행동). 유튜브·쓰레드·릴스·인스타 모두 초기 진입자가 성공한 반복 패턴.
- 카카오톡 간편결제 인프라 → 전환율 높을 것.

#### 네이버 블로그/CUE 트렌드 (출처: 2026년 블로그 전성기, CUE 알고리즘, 2024 블로거 바사삭)
- **블로그 지수 폐지 → 홈판 알고리즘**: 신규 블로그도 콘텐츠 품질 기준 동등 경쟁(2~3편 신규 계정도 홈판 노출, 수만 조회수). 블로그 지수 폐지 = 종말 아닌 신규 진입 기회.
- **CUE(AI 검색)**: 창의적 추론이 아니라 네이버 블로그·지도에서 가장 많이 언급·신뢰도 높게 쌓인 콘텐츠를 수집·요약 → 기존 SEO 연장선. 키워드는 '명사' 단위가 아니라 '사용자 의도 문장' 단위로 발굴(북카페 → 루프탑·조용한·사람 없는·힐링·셀카존). 플랫폼 본질 = 광고 수익 = 체류 시간 → 고품질 콘텐츠는 알고리즘 바뀌어도 항상 유리.

### 일머리·마인드셋·직원관리·커리어

#### 능동성 = 취업·연봉 핵심 (출처: 마케팅 취업 연봉상승 200%)
- 지원자 3등급: 이력서만(탈락) → 이력서+포트폴리오(불충분) → 이력서+포트폴리오+**지원 회사 분석 제안서**(합격).
- 능동성 증명 공식: 회사 → 홈페이지/SNS/콘텐츠 분석 → 개선 제안서(레퍼런스 근거 수치화) → 이력서 첨부/면접 발표.
- 면접 좋은 답변(회사 관점): "제가 들어와서 이렇게 기여할 수 있습니다. 채널 분석해보니 [수치/레퍼런스] 적용하면 개선 가능." / 최악(자기중심): "제가 다니면 이렇게 성장할 것 같습니다."
- 마케터 핵심 자질 = 타인 공감력(자기중심적 사고 = 콘텐츠도 자기 관점만 → 공감 못 얻음 → 1년 미만 퇴사).
- 연봉 차이는 기획력에서: 퍼포먼스 마케터(실행, 2천만 미만) vs 비즈니스 모델 제안 가능 기획형(수억~10억).
- 제안의 완성도보다 제안 행위 자체가 중요(허접해도 분석·제출했다는 사실이 강한 인상). 실증: 다현 님(회사 칭찬 블로그 글 작성 후 링크 첨부 → 합격), 대도서관(할 줄 모르는 기술을 "할 수 있다" 말하고 독학).

#### 대학 진학 판단 (출처: 2024년 이 시대에 대학을 가야 할까)
- 가야 할 유형: 전문직(의사·변호사) 지망, 시킨 것 잘하는 유형, 수행평가 잘하는 유형.
- 가지 않아도 되는 유형: 책 읽는 거 좋아하는 사람, 돈을 많이 벌고 싶은 사람.
- "대학 가야 되나요?" 질문하는 사람 자체가 이미 '가야 할 유형'이 아닐 가능성.
- 정보 비대칭 해소(2000년대 중반 전환점)로 대학 프리미엄 축소. 대학 집착의 뿌리 = 부모 세대의 열등감 전이.
- 수능 공부의 진짜 가치 = 사고력 향상(내신은 머리 안 좋아지지만 수능은 사고력 시험). 실패한 시행착오도 자산.

#### 인사관리 6원칙 + 1 (출처: 직원관리 인사 가스라이팅)
1. **높은 임금 1.2배 원칙** — 시장가 대비 능력의 1.2배 선제 지급(질 좋은 지원자 집중→이탈률↓→교육비·정치질·재채용 비용 방지).
2. **4배수 원칙 교육** — "월급 300만 = 월 1,200만 이상 벌어와야 의미"(수익 구조 투명 공개, 매출≠실수익).
3. **방어기제 없는 인간 뽑기** — 피드백 불수용·변명·정색하는 사람 채용 금지(조직 붕괴 트리거: 피드백 불수용→불평→정치질→대표 왕따→팀장 권위 붕괴).
4. **사회적 지능 높은 팀장 배치** — 실력보다 리더십·사회지능 우선(팀장 1명 잘못 세우면 팀 전체 인재 무력화).
5. **명확한 비전 + 약속 지키기** — 1:1 면담으로 개인 꿈 확인 후 맞춤 비전, 약속은 문서·카톡 기록, 기억 어긋나면 상대 기억 우선 인정.
6. **인사 마케팅** — 채용 공고를 세일즈 페이지처럼 상세·친절하게, 면접도 회사 어필의 장(갑질·"내가 기회 던져준다" 금지).
7. (보너스) **대표의 압도적 실력 유지** — 따라잡히지 않게 지속 학습, 퍼스널 브랜딩(유튜브)으로 가치관 맞는 인재 필터링.
- 비전 유형: ① 돈만 원하는 유형(돈으로 관리) ② 성장·꿈 지향(맞춤 비전+약속). 약속 신뢰 = 간부 충성도 = 조직 안정. 직원 = "귀한 집 자식이자 나와 같은 인간"(톱니바퀴 취급 금지). 조직문화는 직원 표정에서 드러남(경비 아저씨 사례).

#### AI를 직원처럼 활용 (출처: 상위 1% 일잘러 ChatGPT, 나만을 위한 AI, 이제 직원 0명, 클로바노트)
- AI = 채용 대신 쓰는 도구("교육시켜 일하게 만든다"). 자청: GPTs 활용 한 달 월 1억 비용 절감, 매년 20명 채용하던 것을 자동화로 대체.
- GPTs 5분 제작: ChatGPT에 "나는 GPTs 등록할 프롬프트를 만들 거야. 임무는 [역할]이야. 대신 [조건]이어야 해. 프롬프트를 만들어 줘" → 복사 → GPT 만들기 → 지침 붙여넣기 → 발행. 기초(300자) → 반복 수정 → 고도화(8,000자). 조건엔 예시 2~3개 추가.
- 직장인은 AI 답변 기반 기획서를 상사·대표에게 제출 → 높은 평가.
- 회의록 자동화(클로바노트): 녹음 → 텍스트 추출 → ChatGPT(20분 이하)/Claude(20분 초과)에 업로드 → "회의 내용 500자 요약 + 초등학생도 이해하게 500자 + 창의적 아이디어 1000자" 식 용도·분량 구체 지정.
- 프롬프트 능력은 플랫폼 독립적 자산(GPTs 마스터 → Claude·Perplexity 동일 적용).

#### 마인드셋 핵심 원칙
- **자의식 해체** (출처: 베스트셀러 작가, 뷰트랩): 성과 없는 초반에 자기 방식 고집하면 실패. "내가 별로면 아닌데" 하는 크리에이터 자아 버리고 대중이 원하는 걸 채택.
- **첫발 장벽 극복** (출처: 나만을 위한 AI, 나락 자청, 릴스 100만): 릴스·마케팅·온라인 강의·GPT·AI 모두 한 번 해본 사람에게 쉬워짐. "10분만 해보자"로 진입 장벽 낮추기. 인간은 진화적으로 새로운 시도에 거부감(유전자 오작동) → 의식적으로 이겨내기.
- **편안함 = 위험 신호** (출처: 돈 버는 사장): 편안함 느낄 때 오히려 더 달려야.
- **실행 근력** (출처: 5분만에 조회수, 쓰레드): 초보는 당연히 반응 없음 → 10개 이상 올리며 실행 근력 쌓는 과정이 먼저. 일단 시작해야 분석하게 되고 실력이 오른다.
- **레퍼런스 차용** (출처: 다수): 본인 머리를 믿지 말고 잘된 형식을 차용(책 출판·릴스·쓰레드 모두 동일). 형식 모방 ≠ 내용 표절.
- **재능 맹점 오류** (출처: 월1000 개소리): 자신이 성공하면 "누구나 가능"으로 착각 — 다양한 표본을 못 보면 자기 기준을 평균으로 일반화. 돈 버는 것도 재능 영역.
- **기회 비용** (출처: U-I-S): 월 1억짜리 사업 둘 있을 때 월 1,000만 아이디어 추가 실행은 비효율 — 최고 수익원 집중.
